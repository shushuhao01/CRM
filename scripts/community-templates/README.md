## ✨ 项目简介

**云客 CRM** 是一套面向中小企业的**开源全链路客户关系管理系统**，覆盖从客户获取、销售跟进、订单处理、物流追踪、财务结算到售后服务的完整业务流程。完整版拥有 **多端覆盖**（PC Web + H5 移动端 + 微信小程序 + 外呼助手 APP + 企微侧边栏），是同等价位中功能最全面的 CRM 产品。

不同于大多数只做"客户信息管理"的 CRM，云客 CRM 提供了一套真正可以**日常使用的完整业务系统**：你可以用它管理客户资料和跟进记录，也可以用它创建订单、管理商品库存、安排发货、追踪物流轨迹、处理售后退换货，并通过数据看板实时掌握业务全貌。完整版更深度集成**企业微信**（侧边栏 H5、会话存档、获客助手、智能质检），提供**微信小程序**让客户自助填写收货信息（隐私加密、仅授权人员可见），真正做到从获客到成交的私域闭环。

## 功能详情

### 📊 数据看板

系统首页提供可视化数据看板，一目了然掌握业务全貌。

- **核心数据卡片** — 今日新增客户数、新增订单数、成交金额、待处理事项等关键指标实时展示
- **销售趋势图表** — 按日/周/月维度展示订单量、成交额趋势，支持时间范围筛选
- **客户来源分析** — 饼图/柱状图展示各渠道客户数量占比，帮助评估获客效果
- **订单状态分布** — 直观显示各状态订单数量（待确认/已确认/已发货/已完成/已取消）
- **待办事项提醒** — 今日需跟进客户、待审核订单、待发货订单等待办清单

### 👥 客户管理

完善的客户全生命周期管理，帮助你高效管理客户资源。

- **客户信息管理** — 支持创建、编辑、查看、删除客户，包含姓名、电话、邮箱、地址、来源渠道等完整客户资料
- **客户标签与分组** — 自定义标签体系，支持按标签、来源、地区等多维度对客户进行分类和筛选
- **跟进记录管理** — 每个客户支持添加多条跟进记录，记录沟通内容、跟进时间、下次联系计划
- **跟进提醒** — 设置跟进提醒时间，到期自动提醒，避免遗漏重要客户
- **客户全景画像** — 单个客户页面聚合展示基本信息、所有订单、物流状态、跟进记录、售后记录，一个页面了解客户全部情况
- **数据导入导出** — 支持 Excel 批量导入客户数据、导出客户列表（社区版每次100条）
- **客户分享与转移** — 将客户分享给同事协作跟进，或转移客户归属权
- **回收站** — 删除的客户进入回收站，支持恢复，避免误删
- **高级筛选** — 支持按创建时间、标签、来源、跟进状态、归属人等多条件组合筛选
- **自定义字段** — 可根据行业需求自定义客户字段（如"微信号"、"生日"、"客户等级"等）

### 📦 订单管理

从下单到完成的完整订单生命周期管理，支持多种业务场景。

- **订单创建与编辑** — 创建订单时关联客户、选择商品、设置价格和收货地址，支持多商品明细
- **订单状态流转** — 支持完整的状态流转：待确认 → 已确认 → 已发货 → 已签收 → 已完成，每个状态变更自动记录
- **订单审核工作流** — 新订单可设置审核流程，管理员审核后才能进入后续流程
- **批量审核** — 支持勾选多个待审核订单一键批量通过/驳回
- **退款管理** — 支持订单退款申请、审批、退款完成的完整退款流程
- **订单搜索** — 按订单号、客户名、手机号、下单时间等快速搜索
- **订单详情** — 完整展示订单信息、商品明细、收货地址、物流信息、操作日志
- **订单备注** — 订单支持内部备注，方便团队沟通协作

### 🚚 物流管理

对接主流快递公司，实时追踪物流轨迹，高效管理发货流程。

- **发货管理** — 订单确认后可直接选择物流公司、填写运单号进行发货操作
- **物流轨迹追踪** — 实时查询物流轨迹信息（支持快递100等API），自动更新物流状态
- **快递公司管理** — 系统预置 30+ 家主流快递公司（顺丰、中通、圆通、韵达、极兔等），可自行添加
- **寄件人地址管理** — 维护多个寄件人/退货地址，发货时快速选择
- **面单打印** — 支持快递电子面单打印，一键生成面单标签
- **批量发货** — 支持 Excel 批量导入运单号，批量操作发货
- **物流状态自动更新** — 系统自动同步最新物流状态，签收后自动更新订单状态

### 🛍️ 商品管理

支持**实物商品 + 虚拟商品 + 服务套餐**三类商品类型，满足多元业务场景。

- **商品 CRUD** — 创建、编辑、上下架、删除商品，支持商品图片、规格描述
- **商品分类** — 多级商品分类管理，支持按分类浏览和筛选商品
- **库存管理** — 实时库存数量管理，下单自动扣减库存、退货自动回补
- **库存预警** — 设置库存预警阈值，库存不足时自动提醒
- **虚拟商品 — 卡密管理**（完整版）— 支持卡密类商品（充值卡、激活码、会员卡、软件 License 等），客户下单后自动发放卡密，无需人工发货
- **虚拟商品 — 资源库存**（完整版）— 支持文件/链接类资源商品（课程视频、电子书、设计素材、源码包等），购买后自动开通下载/查看权限
- **服务类套餐**（完整版）— 支持无需发货的服务商品（咨询服务、年费会员、SaaS 订阅、培训课程等），订单确认即完成，无物流环节
- **商品搜索** — 按商品名称、分类、SKU、商品类型等快速搜索

### 🔧 售后管理

完整的售后服务流程，提升客户满意度。

- **售后工单** — 客户申请退货/换货/维修时创建售后工单，记录问题描述和处理方案
- **售后状态流转** — 待处理 → 处理中 → 已完成/已关闭，每个环节可添加处理记录
- **退货/换货管理** — 支持退货退款、换货补发等不同售后类型
- **售后统计分析** — 售后数量趋势、售后类型分布、处理时效统计，帮助优化服务质量
- **关联订单** — 售后工单自动关联原始订单，方便查看订单详情和物流信息

### ⚙️ 系统管理

安全可靠的系统管理功能，满足团队协作需求。

- **部门管理** — 创建企业组织架构，支持部门层级
- **用户管理** — 添加/编辑/禁用用户账号，设置用户所属部门和角色（社区版 ≤3 用户）
- **角色权限管理** — 基于 RBAC 模型的权限系统，可自定义角色和权限分配
- **操作日志审计** — 记录所有关键操作（登录、数据变更等），方便安全审计和问题追踪
- **消息通知 & 实时推送** — 系统内消息通知 + 实时推送到**企业微信/钉钉**（新订单、审批提醒、客户动态等关键事件实时触达）
- **资料分配** — 管理员将已签收订单的客户资料批量分配给销售成员跟进，支持待分配/已分配/已归档状态管理
- **客户分享** — 将客户分享给同事协作跟进，支持设置分享有效期，过期自动回收权限
- **自定义字段管理** — 管理客户自定义字段的定义和配置
- **授权管理** — 输入授权码激活系统，查看授权状态和到期时间
- **资料列表** — 系统基础数据维护（如客户来源、标签预设等）

### 📱 H5 移动端

随时随地处理业务，不受办公地点限制。

- **移动端客户管理** — 手机上查看客户列表、客户详情、添加跟进记录
- **移动端订单查看** — 查看订单列表和详情、物流信息
- **响应式适配** — 自动适配各种屏幕尺寸，移动体验流畅

## 完整版 CRM 功能全景

> 以下模块为**完整版（专业版/企业版/私有部署版）**独有或增强功能。社区版专注核心 CRM 流程，完整版在此基础上覆盖企业级全场景需求。

### 💼 财务管理

完整的财务核算与结算体系，连通订单、物流、代收与外包结算。

- **代收管理** — 已发货订单的快递代收货款（COD）追踪，统计订单金额/需代收金额/已返款/未返款，排除虚拟商品订单
- **改代收申请与审核** — 支持修改代收金额申请（取消代收/减少代收），上传尾款凭证，管理员审核通过/拒绝
- **返款管理** — 快递代收货款返还追踪，已返款/未返款状态管理，按部门/人员/日期维度统计
- **增值管理（外包费用结算）** — 外包公司管理、订单分配到外包公司、单价阶梯配置、有效/无效订单审核、结算报表生成
- **财务结算报表** — 多维度报表导出，支持按外包公司/人员/部门/时段生成结算单

### 📈 业绩与绩效管理

销售业绩多维度统计与提成计算，驱动团队目标达成。

- **下单业绩** — 统计成员下单数量与金额（排除取消/拒绝/退款等无效订单）
- **签收业绩** — 统计已签收订单的单数与金额，真实反映回款能力
- **发货业绩** — 统计已发货订单数量与金额，含在途/签收/拒收明细
- **团队业绩排行** — 部门维度成员业绩排名，支持按下单额/签收额/发货额排序
- **业绩分享** — 多人分单场景，一笔订单按比例分配给多个成员，自动按比例计算各自提成金额
- **绩效提成计算** — 支持按下单单数或签收业绩金额设置阶梯提成比例，自动计算成员提成

### 📞 通话管理

对接主流云通信平台，构建销售电话中台。

- **通话记录管理** — 自动记录每通电话的时间、时长、通话结果，关联客户/订单
- **通话录音存储** — 全部通话自动录音、云端存储，支持在线回放和下载
- **坐席状态管理** — 实时查看每个坐席的在线/忙碌/空闲状态
- **外呼线路配置** — 支持配置多条外呼线路（阿里云通信等），按策略分配
- **电话外呼系统集成** — 一键外呼、来电弹屏、通话中记录跟进，无需切换工具
- **工作手机绑定** — 绑定销售工作手机号，通话记录自动同步至 CRM

### 💬 短信管理

智能短信触达，高效维护客户关系。

- **短信模板管理** — 预设多种业务场景短信模板（发货通知、催付提醒、节日问候等）
- **短信发送记录** — 完整的发送历史记录，查看送达状态和回执
- **自动发送规则** — 设置触发条件（如订单发货、客户生日），自动发送对应短信
- **短信额度管理** — 短信包购买、额度查询、用量统计
- **短信审批** — 群发短信需经管理员审批，防止误发

### 🏢 企业微信深度集成

**30+ 企微子功能**，打通企微私域与 CRM 业务数据，是同价位产品中企微集成深度最强的 CRM。

#### 企微侧边栏 H5

在企微聊天窗口右侧嵌入 CRM 侧边栏应用，销售与客户沟通时**无需切换工具**即可：

- 实时查看当前聊天客户的 **360° 画像**（基本信息、订单记录、跟进历史）
- 一键创建跟进记录、新建订单、发送商品卡片
- 查看客户标签、历史购买偏好，辅助精准推荐
- 调用话术库快速回复，提升沟通效率

#### 会话存档

合规留存企微内外部聊天记录，支持监管审计：

- **聊天记录全量归档** — 自动拉取并存储员工与客户的聊天记录（文字、图片、文件、语音等）
- **敏感词监控** — 预设敏感关键词（如"飞单""私下转账"），触发时自动告警
- **AI 智能质检** — AI 分析会话内容，评估服务质量、识别商机和风险
- **合规审计** — 管理层可按时间/人员/关键词检索历史会话，满足企业合规要求

#### 获客助手与活码

企微获客的核心工具，高效承接流量：

- **渠道活码** — 为不同推广渠道生成专属二维码，客户扫码自动分配给对应销售
- **自动打标签** — 根据扫码渠道/活动自动为客户打上来源标签
- **欢迎语** — 客户添加后自动发送个性化欢迎语和资料

#### 其他企微能力

- **客户同步** — 企微通讯录客户自动同步到 CRM，双向数据一致
- **客户群管理** — 管理企微客户群，群模板批量建群，群消息统计
- **群模板** — 预设群名称、群公告、群欢迎语模板，一键创建标准化客户群
- **对外收款** — 在企微会话中直接发起收款，收款记录自动关联订单
- **AI 智能助手** — AI 辅助回复客户问题，基于知识库智能推荐答案
- **企微多企业授权** — 支持绑定多个企业微信主体，集团化管理

### 📲 微信小程序（客户端）

让客户通过微信小程序自助完成信息填写，**隐私安全、体验优雅**。

- **客户自助填写** — 销售通过企微/微信发送小程序卡片，客户打开后自助填写收货地址、联系方式等信息
- **隐私加密存储** — 客户填写的手机号、地址等敏感信息**加密存储**，仅授权销售人员可查看，其他人员不可见
- **表单灵活配置** — 管理员可配置小程序表单字段（如姓名、电话、地址、备注等），按业务需求灵活调整
- **自动关联客户** — 客户提交后信息自动回填到 CRM 客户资料中，销售无需手动录入
- **发送记录追踪** — 记录每次小程序卡片的发送和客户提交情况，追踪转化效果

> **为什么这个功能很重要？** 传统方式让客户口述地址容易出错，拍照发送又涉及隐私泄露。通过小程序填写，客户信息直接加密进入系统，销售无需接触原始敏感数据，既安全又高效。

### 📱 外呼助手 APP（Android + iOS）

专为销售外呼场景打造的移动端辅助工具，支持 Android 和 iOS 双平台。

- **一键外呼** — 在 APP 中直接拨打客户电话，通话记录自动同步至 CRM
- **通话记录同步** — 每通电话的时间、时长、结果自动记录并关联客户
- **客户速查** — 外呼前快速查看客户基本信息和跟进历史
- **消息推送** — 新线索分配、跟进提醒等消息实时推送到手机

### 🔗 开放平台能力

为系统集成和二次开发提供完善的接口能力。

- **RESTful API** — 1400+ API 端点全面开放，文档覆盖 29 章节
- **WebSocket 实时推送** — 订单状态变更、新消息等事件实时推送到前端
- **Webhook 回调** — 支持配置回调地址，关键业务事件（新订单、客户变更等）自动通知外部系统
- **API 鉴权与限流** — 基于 Token 的 API 鉴权 + 速率限制，保障接口安全

#### 环境要求

- **Node.js** >= 18
- **MySQL** >= 8.0
- **npm** >= 9

#### 步骤

```bash
# 1. 克隆代码
git clone https://github.com/shushuhao01/yunke-crm.git
cd yunke-crm

# 2. 初始化数据库
# 在 MySQL 中创建数据库，然后导入 schema
mysql -u root -p -e "CREATE DATABASE yunke_crm CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

cd ..                  # 回到项目根目录
cp .env.example .env   # 编辑 API 地址
npm install
npm run dev            # 启动前端（开发模式）

# 5. 访问 http://localhost:5173
```

| **电商/跨境私域** | 订单多、发货杂、客户跟进难 | 订单+物流+售后全流程 + 企微私域运营 |
| **教育培训** | 学员跟进、课程推荐、续费管理 | 客户标签 + 跟进提醒 + 业绩考核 + 小程序收集信息 |
| **本地生活服务** | 客户分散、沟通低效、数据不通 | 企微侧边栏 + 会话存档 + 微信小程序 |
| **外贸公司** | 客户跟进周期长、物流追踪复杂 | 客户画像 + 跟进记录 + 物流实时追踪 |
| **大健康/医美** | 客户隐私敏感、复购管理、会员跟进 | 小程序隐私填写 + 客户标签 + 跟进提醒 + 企微私域运营 |
| **知识付费/在线教育** | 虚拟商品交付自动化、课程权限管理 | 卡密自动发放 + 资源自动开通 + 服务套餐订单即完成 |
| **游戏/软件/SaaS** | 激活码发放、订阅管理、客户续费 | 卡密库存管理 + 自动发放 + 业绩统计 + 客户跟进 |
| **会员卡/充值卡销售** | 卡密库存混乱、发放效率低 | 卡密库存批量导入 + 下单自动发放 + 发放记录可追溯 |
| **微型创业团队** | 预算有限，需要一套完整工具 | 社区版永久免费，3 人够用 |
| **系统集成商/ISV** | 需要 CRM 底座做二次开发 | 完整源码 + 清晰分层架构 + Apache 2.0 协议 |
| **政企/金融/医疗** | 数据不出内网、合规审计 | 私有部署 + 数据100%私有 + 会话存档 + 操作日志审计 |
| **开发者学习** | 学习 Vue 3 + Node.js 全栈项目 | 45万+行工程级代码、最佳实践参考 |
