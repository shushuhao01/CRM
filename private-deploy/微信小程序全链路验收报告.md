### 5.2 CRM租户是否需要额外配置小程序ID？

**不需要。** 小程序属于服务商（运营方），所有租户共用同一个小程序。

- **AppID/AppSecret** 在 Admin 管理后台统一配置
- **租户端** 只需在「企微管理→侧边栏→小程序配置」中设置卡片标题和海报等展示信息
- 转发卡片时，后端自动从 Admin 全局配置读取 AppID

| 域名类型 | 填写内容 | 说明 |
|----------|----------|------|
| **request 合法域名** | `https://你的CRM后端域名` | 例如 `https://crm-api.yunkes.com`，**必须是 https** |
| uploadFile 合法域名 | 不需要 | 本小程序无文件上传 |
| downloadFile 合法域名 | 不需要 | 本小程序无文件下载 |
| socket 合法域名 | 不需要 | 本小程序无 WebSocket |
| udp 合法域名 | 不需要 | — |

**注意事项**：
- 域名必须已备案（ICP），不能使用 IP 地址
- 必须是 `https://` 协议，不支持 `http://`
- 每月修改上限 **50 次**（别填错反复改）
- 部署时同步修改 `miniprogram/app.js` 中的 `apiBaseUrl`

#### 步骤三：IP 白名单配置

> 路径：「开发管理 → 开发设置 → 服务器域名 → IP白名单」（下方区域）

本小程序使用了 `getPhoneNumber` 获取微信手机号，后端需调用微信 API (`api.weixin.qq.com`) 获取 `access_token` 和解密手机号。**必须将服务器出口 IP 加入白名单**：

| 填写内容 | 说明 |
|----------|------|
| 你的服务器公网 IP | 宝塔面板 → 首页可看到，例如 `120.79.xxx.xxx` |
| 多台服务器则逐行填入 | 每行一个 IP |

**如何确认服务器出口 IP**：
```bash
# 在服务器上执行
curl ifconfig.me
```

**不配置的后果**：调用微信 API 时返回 `errcode: 40164, invalid ip xxx not in whitelist`

## 六、微信流量主广告接入指南

### 6.1 开通条件

| 条件 | 要求 |
|------|------|
| 小程序类目 | 非游戏类 |
| 累计独立访客 | ≥ **1000 UV**（近 30 天） |
| 无严重违规 | 无虚假宣传/恶意刷量记录 |

### 6.2 开通步骤

1. 登录 [mp.weixin.qq.com](https://mp.weixin.qq.com) → 左侧菜单「流量主」→ 开通
2. 创建广告位 → 选择 **Banner 广告** → 获得 `adUnitId`（如 `adunit-1a2b3c4d5e6f7g8h`）
3. 将 `adUnitId` 填入 `miniprogram/pages/success/success.js`：

```javascript
data: {
  adUnitId: 'adunit-你的广告位ID'  // ← 替换这里
}
```

4. 提交审核发布后，广告即自动展示

### 6.3 广告放置策略

| 页面 | 是否有广告 | 原因 |
|------|----------|------|
| 表单填写页 (form) | ❌ 无 | 避免干扰填写体验 |
| 提交成功页 (success) | ✅ **有** | 用户已完成任务，低干扰 |
| 链接过期页 (expired) | ❌ 无 | 访问量低，无意义 |

### 6.4 广告类型选择

| 类型 | ad-type 值 | 收益 | 适合场景 |
|------|-----------|------|----------|
| **Banner 广告**（当前使用） | `banner` | 中等 (10-50 元/千次) | ⭐ 推荐，干扰最小 |
| 格子广告 | `grid` | 中等 | 多图标展示 |
| 激励视频 | `video` | 较高 | 需用户主动点击，不适合此场景 |

### 6.5 收益结算

- **结算方式**：eCPM（每千次曝光收入）
- **提现**：每月微信自动结算到对公账户
- **查看数据**：mp.weixin.qq.com → 流量主 → 数据报表
