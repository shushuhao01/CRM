## 一、产品概述：云客CRM 是什么？

### 1.1 一句话介绍

**云客CRM** 是一套面向中小企业的全链路智能销售管理系统。它帮您把「获客→跟进→成交→发货→收款→售后」这整条销售链路，全部管起来，全部看得见。

### 1.2 解决什么问题？

| 您可能遇到的困扰                | 云客CRM 怎么解决            |
| ----------------------- | --------------------- |
| 客户资料散落在 Excel、微信、纸质笔记本里 | 统一客户档案，手机/电脑随时随地查看    |
| 销售离职带走客户                | 客户归属到人，离职一键转移，客户不流失   |
| 订单审核靠口头沟通，容易出错          | 线上审核工作流，每一步都有记录       |
| 不知道发了多少货、收了多少钱          | 物流轨迹实时追踪，COD代收自动对账    |
| 销售业绩算不清楚                | 按人/按部门/按时间自动统计，佣金阶梯计算 |
| 企业微信客户不知道怎么管            | 企微深度对接，客户自动同步、聊天记录存档  |

### 1.3 适用哪些行业？

虽然云客CRM是通用型CRM，但在以下行业表现尤为出色：

- 🏥 **健康美业**：健身、医美、保健品、健康管理
- 📦 **零售电商**：实物商品销售、代收货款
- 💊 **医药健康**：客户健康档案、改善目标跟踪
- 📱 **私域运营**：企业微信获客+社群运营
- 💻 **虚拟商品**：卡密发货、网盘资源交付

### 1.4 核心亮点速览

| 亮点             | 说明                                  |
| -------------- | ----------------------------------- |
| 🖥️📱 **多端协同** | PC端 + H5移动端 + 微信小程序 + 手机APP + 企微侧边栏 |
| 🔗 **全链路闭环**   | 获客→跟进→成交→发货→收款→售后 一站式管理             |
| 🛡️ **数据安全**   | 客户资料加密、手机号脱敏、操作留痕、权限分级              |
| 📊 **数据可视化**   | 仪表盘图表、销售漏斗、业绩排行，经营状况一目了然            |
| 💰 **COD代收**   | 货到付款场景的完整解决方案，代收返款自动对账              |
| 🚚 **物流管理**    | 9家快递公司对接，轨迹实时追踪，面单批量打印              |
| 💬 **企微深度集成**  | 客户自动同步、会话存档、AI质检、活码获客               |

### 2.2 数据安全保障

您的数据安全是我们的头等大事，云客CRM从多个层面保护您的业务数据：

#### 🔐 访问安全

- **HTTPS加密传输**：所有数据在网络中都是加密的，防止被窃听
- **JWT令牌认证**：每次操作都验证身份，令牌过期自动要求重新登录
- **登录异常检测**：连续输错密码自动锁定账户，防暴力破解
- **IP白名单**：可设置员工只能在公司网络登录

#### 👁️ 权限管控（谁可以看到什么）

系统支持四级精细权限：

| 级别       | 控制内容     | 举例                     |
| -------- | -------- | ---------------------- |
| **菜单权限** | 哪些功能模块可见 | 销售看不到财务模块              |
| **操作权限** | 能做什么操作   | 只能看订单，不能删除             |
| **数据权限** | 能看到哪些数据  | 销售只看自己的客户，经理看全部门       |
| **字段权限** | 能看到多详细   | 手机号显示为 138\*\*\*\*8888 |

#### 🗄️ 数据保护

- **密码加密存储**：无人可以查看到用户原始密码
- **敏感信息脱敏**：客户手机号、身份证等自动部分隐藏
- **操作日志全记录**：谁、在什么时间、做了什么操作，全部可追溯
- **数据定期备份**：防止意外数据丢失

#### 🏢 两种部署模式满足不同需求

| 对比维度     | 云端版 (SaaS) | 私有部署版       |
| -------- | ---------- | ----------- |
| **适合企业** | 大多数中小企业    | 对数据有特殊合规要求的 |
| **服务器**  | 我们提供云端服务器  | 部署在您自己的服务器上 |
| **维护**   | 我们负责运维升级   | 您的IT团队维护    |
| **开通速度** | 注册即用，5分钟上手 | 需要部署，1-2天   |
| **数据存放** | 云端安全存储     | 完全在企业内部     |

## 三、快速上手：注册与登录

### 3.1 第一步：访问官网

打开浏览器，访问官方网站：**<https://www.yunkes.com>**

在官网您可以：

- 查看产品功能介绍
- 对比不同版本套餐
- 申请**30天免费试用**（全功能开放）
- 在线注册账号

### 3.2 第二步：注册账号

1. 在官网点击「**免费注册**」按钮
2. 填写您的企业基本信息（公司名称、手机号等）
3. 选择适合的套餐版本（可先选免费试用）
4. 完成注册后，系统会自动为您创建企业账号

### 3.3 第三步：登录会员中心

注册成功后，您可以在**会员中心**（官网右上角「登录」）进行以下管理：

- 📋 查看当前套餐与到期时间
- 👥 管理企业信息
- 💳 查看账单与续费
- 📦 购买扩容包（增加用户数/存储空间）
- 🔑 查看授权信息

### 3.4 第四步：登录CRM系统

1. 打开CRM系统地址（注册后会提供给您）
2. 使用管理员账号登录（注册时设置）
3. 首次登录后，建议按顺序完成以下设置：
   - ① 创建**部门**（如销售一部、销售二部）
   - ② 添加**员工账号**并分配部门和角色
   - ③ 在「系统设置」中完善企业基本信息
   - ④ 导入现有客户数据（支持Excel批量导入）

### 3.5 第五步：邀请团队

在「系统设置 → 用户管理」中添加团队成员账号，分配好角色后，团队成员即可登录使用。

**常见角色及权限一览**：

| 角色        | 能看到什么       | 适合谁    |
| --------- | ----------- | ------ |
| **超级管理员** | 全部数据和功能     | 老板/总经理 |
| **管理员**   | 全部数据、多数管理功能 | 销售总监   |
| **部门经理**  | 本部门全部数据     | 销售经理   |
| **销售人员**  | 自己的客户和订单    | 一线销售   |
| **财务**    | 财务相关数据      | 财务人员   |
| **客服**    | 客户资料（脱敏）+售后 | 客服人员   |

## 五、客户管理

> **这是CRM系统的核心模块**，管理您最重要的资产——客户。

### 5.1 客户列表

#### 📋 列表功能

客户列表是您日常最常使用的页面，提供以下能力：

- **搜索与筛选**：按姓名、手机号、客户等级、跟进状态、创建时间等条件快速找到客户
- **关键词搜索**：一个搜索框同时搜索姓名和手机号
- **批量操作**：勾选多个客户，可批量分享、转移、导出Excel、删除标签
- **自定义列**：自由选择表格中显示哪些列，符合您的工作习惯
- **分页浏览**：支持翻页、跳转，管理海量客户不卡顿

#### 🔒 数据权限

每个销售人员登录后，看到的客户是不同的：

- **普通销售**：只能看到自己创建的客户 + 分配给自己的客户 + 别人分享给自己的客户
- **部门经理**：可以看到本部门所有成员的客户
- **管理员/老板**：可以看到全公司的客户

这样既保护了客户资源，又保证了管理的透明度。

手动创建客户分组（如「VIP客户群」「待跟单客户」），将客户归类管理：

- 创建分组
- 查看分组下的客户
- 批量移动客户到其他分组

### 5.7 客户分享

> **特色功能**：当客户需要多人协作跟进时，可以分享客户给同事。

**操作流程**：

1. 在客户列表勾选要分享的客户
2. 点击「批量分享」（或单个客户详情页点击「分享」）
3. 选择接收人
4. 设置**时间限制**：X天后自动回收，或永久分享
5. 填写备注（可选）

**分享规则**：

- 分享后，接收人可以看到该客户的全部信息
- 到了设定的时间，客户自动归还给原归属人
- 分享人可随时「手动回收」

### 5.8 客户转移

当销售离职或客户需要换人跟进时，可以使用转移功能：

1. 选择要转移的客户
2. 选择新归属人
3. 系统自动记录分配历史（谁→谁、操作人、时间、原因）

### 5.9 Excel 批量导入

如果您有大量历史客户数据：

1. 下载系统提供的 **Excel导入模板**
2. 按模板格式填写客户数据
3. 上传文件，系统自动校验
4. 预览数据，确认无误后一键导入
5. 系统自动检测重复客户（按手机号去重）

## 六、订单管理

> **从创建到审核，从发货到签收，从收款到结算——订单全生命周期管理。**

### 6.1 订单全生命周期

一张订单从创建到完成，会经历以下状态流转：

```
                       创建订单
                          ↓
                    待流转（系统自动）
                          ↓
                    待审核 ═══════ 审核拒绝 → 退回修改
                      ↓
                   审核通过
                      ↓
                    待发货
                      ↓
                    已发货 ──── 拒收 → 退回
                      ↓
                    已签收
                      ↓
                    已完成
```

**其他状态路径**：

- 已发货后可**取消订单** → 待取消 → 已取消 → 已退款
- 签收后可**申请售后** → 已建售后
- 物流途中可**包裹异常**

### 6.2 订单列表

订单列表页面支持：

- **多维度筛选**：订单号、客户姓名/电话、订单状态、支付状态、日期范围
- **彩色状态标签**：21种订单状态一目了然
- **快捷操作**：查看详情、审核（经理）、发货、取消、打印
- **待办标记**：重要订单设为待办，设置提醒日期

### 6.3 创建订单

点击「新增订单」，进入订单创建页面：

#### 第一步：选择客户

从客户列表中选择下单客户（必选），客户的收货地址、电话自动带入。

#### 第四步：提交订单

### 6.4 订单审核

## 九、财务管理

### 9.1 COD代收管理（货到付款）

> **独创功能**：专为"货到付款"销售模式设计。

#### COD是什么？

COD = Cash On Delivery（货到付款）。客户收货时付款给快递员，快递公司再结算给您。云客CRM帮您完整管理这个流程。

#### COD完整流程

```
1. 创建订单 → 设置COD金额（如¥299）
       ↓
2. 审核通过 → 发货
       ↓
3. 快递员送货 → 向客户收款¥299
       ↓
4. 客户签收 → 快递公司定期结算给您
       ↓
5. 财务操作：查看「代收管理」→ 操作「返款」
   → 记录：已返款¥299，返款时间，返款人
```

#### 代收管理功能

| 功能         | 说明              |
| ---------- | --------------- |
| **代收列表**   | 查看所有COD订单，按状态筛选 |
| **返款操作**   | 标记快递公司已结算付款     |
| **改代收**    | 修改代收金额          |
| **取消代收申请** | 特殊情况申请取消代收      |
| **取消代收审核** | 审核人审批           |

### 9.2 绩效数据

#### 数据统计

财务模块提供绩效数据概览：

- 发货单数
- 签收单数
- 有效单数
- 有效系数合计
- 预估佣金合计

#### 佣金阶梯

管理员可设置佣金阶梯规则，例如：

| 月业绩             | 提成比例 |
| --------------- | ---- |
| 0 - 5,000       | 3%   |
| 5,001 - 10,000  | 5%   |
| 10,001 - 20,000 | 8%   |
| 20,001 以上       | 10%  |

系统根据规则自动计算每个销售的预估佣金。

### 9.3 结算报表

按时间段、部门、人员维度查看结算数据，支持导出Excel。

## 十二、资料管理

> **订单签收后，客户资料自动进入资料库。统一管理、按需分配、封存归档、回收站保护 — 让每一条客户资料都有序流转。**

### 12.1 什么是资料管理？

当客户在CRM中完成订单并签收后，该客户的资料会自动进入「资料库」。资料管理的核心逻辑是：**把已成交的客户资料统一收纳、按需分配给销售进行后续跟进（复购、售后关怀等），避免资料散落和销售撞单。**

#### 资料从哪里来？

| 来源 | 说明 |
|------|------|
| **订单签收自动转入** | 客户签收后，资料自动进入资料列表，状态为「待分配」 |
| **手动录入** | 管理员/经理可直接在资料管理中新增资料 |

### 12.2 资料列表

进入「资料管理 → 资料列表」，可以看到所有已签收客户资料的汇总视图。

#### 📊 汇总数据卡片

页面顶部展示四个核心指标：

| 指标 | 含义 |
|------|------|
| **资料总数** | 系统中所有资料的总量 |
| **待分配** | 尚未分配给任何销售的资料数量 |
| **已分配** | 已经分配给销售跟进的资料数量 |
| **订单总额** | 所有资料的订单金额合计 |

#### 🔍 快速筛选

一键切换查看不同时间段的资料：

- 今天 / 昨天 / 本周 / 最近30天 / 本月 / 今年 / 全部

#### 📋 列表筛选与搜索

支持多维度精确查找：

- **状态筛选**：待分配 / 已分配 / 已封存 / 已回收
- **部门筛选**：按部门查看资料
- **归属人筛选**：按分配给的销售查看
- **关键词搜索**：搜索客户姓名、手机号、订单号等

#### 📑 资料记录包含的信息

每条资料记录都关联了完整的业务数据：

- 客户姓名、手机号（脱敏显示）、客户编码
- 关联订单号、订单金额、下单日期、签收日期
- 当前状态（待分配 / 已分配 / 已封存 / 已回收）
- 分配给谁、分配时间
- 操作轨迹（每一次分配、封存、回收的记录）

### 12.3 资料分配

> **核心操作**：将资料分配给具体销售人员，让他们进行后续跟进。

#### 分配方式

| 分配方式 | 说明 | 适用场景 |
|---------|------|---------|
| **指定分配** | 手动选择分配给某个人 | 灵活分配，想给谁就给谁 |
| **轮流分配** | 系统按部门成员轮流分配 | 公平分配，避免厚此薄彼 |
| **随机分配** | 系统随机分配给部门成员 | 快速分配大量资料 |
| **经理分配** | 部门经理统一分配给自己部门成员 | 经理管控分配节奏 |

#### 操作流程

1. 在资料列表中勾选一条或多条资料
2. 点击「分配」按钮
3. 选择分配方式（指定/轮流/随机）
4. 选择目标销售
5. 填写分配备注（可选）
6. 确认分配

分配后，该销售会在自己的客户列表中看到这条客户资料，可以开始跟进。

#### 📊 分配统计

系统提供分配统计视图，帮助管理者了解：
- 每个销售被分配了多少资料
- 今日/本周/本月分配数量
- 平均订单金额
- 最后分配时间

### 12.4 资料封存

> 部分资料暂时不需要跟进（如客户明确表示短期内不购买），可以封存起来，等到合适时机再解封。

#### 封存操作

1. 选择要封存的资料
2. 填写封存原因和封存时长
3. 设置解封时间（自动解封）
4. 确认封存

封存的资料状态变为「已封存」，不再出现在待分配列表中。

#### 解封

到了设定的解封时间，系统会自动解封。也可以手动提前解封，资料重新回到「待分配」状态。

### 12.5 客户查询

> **万能搜索入口**：一个搜索框，查遍所有客户相关数据。

在「资料管理 → 客户查询」中，输入任意关键字即可查找：

**支持的搜索类型**：
| 输入内容 | 匹配结果 |
|---------|---------|
| 客户姓名 | 匹配到该姓名相关的客户 |
| 手机号 | 精确匹配手机号 |
| 客户编码 | 按系统编码查找 |
| 订单号 | 查找该订单对应的客户 |
| 物流单号 | 通过运单号反查客户 |
| 售后单号 | 通过售后工单查找客户 |

**搜索结果展示**：
- 客户基本信息（姓名、编码、标签）
- 匹配类型说明（是通过什么匹配到的）
- 关联的订单信息
- 点击可跳转到客户详情

### 12.6 回收站

> **删除保护**：客户资料被删除后，会先进入回收站保留一段时间，防止误删。

#### 📊 汇总卡片

| 指标 | 含义 |
|------|------|
| **回收总数** | 回收站中的资料总量 |
| **近7天删除** | 最近一周被删除的资料数 |
| **即将过期** | 即将被永久清除的资料数 |

#### 🔍 筛选方式

- **关键词搜索**：搜索客户姓名、电话
- **删除时间**：按今天/近7天/近30天筛选
- **删除人**：按操作人筛选

#### ♻️ 恢复操作

1. 勾选要恢复的资料
2. 点击「恢复」按钮
3. 确认后，资料回到资料列表中

支持**批量恢复**，一次恢复多条资料。

#### 🗑️ 永久删除

1. 勾选要永久删除的资料
2. 点击「永久删除」
3. 二次确认后彻底清除

#### ⏱️ 清理过期

系统会自动清理超过保留期限的回收站资料，也可以手动点击「清理过期」一键清除。

## 十八、常见问题（FAQ）

### Q1：数据安全吗？客户资料会不会泄露？

**A**：非常安全。我们采用多重安全措施：

- 数据传输全程HTTPS加密
- 密码使用不可逆算法加密存储（我们自己也无法看到）
- 手机号等敏感信息在界面上自动脱敏显示
- 所有操作都有日志记录，可追溯
- 如需更高安全性，可选择**私有部署版**，数据完全在您自己的服务器上

### Q2：可以试用吗？试用多久？

**A**：可以！我们提供 **30天全功能免费试用**。访问官网注册即可，无需绑定支付方式。试用期间不限制功能。

### Q3：支持多少人同时使用？

**A**：根据您购买的套餐不同：

- 基础版：10人
- 专业版：50人（推荐）
- 企业版：200人
- 私有部署版：不限

在会员中心可以随时购买扩容包，增加用户数。

### Q4：销售离职了，客户怎么办？

**A**：使用「客户转移」功能，将离职销售的客户一键转移给其他人。所有转移记录都有历史，不用担心客户数据丢失。

### Q5：历史客户数据怎么导入？

**A**：使用Excel批量导入功能。下载系统模板 → 按格式填入客户数据 → 上传即可。支持姓名、手机号、地址等全字段导入。

### Q6：可以打印快递面单吗？

**A**：可以。系统支持电子面单打印，对接顺丰、圆通等主流快递公司，支持Lodop和USB打印。

### Q7：企业微信怎么对接？

**A**：在CRM系统中进入「企微管理」模块，按照指引完成授权配置即可。详细步骤可参考帮助文档或联系技术支持。

### Q8：系统可以定制吗？

**A**：部分功能支持自定义：

- 客户自定义字段（添加您需要的特殊字段）
- 订单自定义字段
- 自定义角色权限
- 自定义订单审核流程

如需深度定制，请联系我们沟通需求。

### Q9：出现问题怎么办？

**A**：我们提供多渠道技术支持：

- 系统内「帮助中心」：覆盖所有模块的操作指南
