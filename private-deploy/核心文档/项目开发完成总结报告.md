## 一、项目总览

云客CRM是一套**多租户 SaaS + 私有部署**双模式的企业级客户关系管理系统，集成了电商履约、企业微信SCRM、电销外呼、物流管理等能力。项目采用前后端分离的全栈架构，包含6大子系统，覆盖PC端、移动端、H5、小程序等多终端。

### 1.1 系统矩阵

| 子系统 | 目录 | 定位 | 技术栈 |
|--------|------|------|--------|
| **CRM主系统** | `src/` + `backend/` | 租户端核心业务平台 | Vue 3 + Express + MySQL |
| **管理后台** | `admin/` | SaaS运营管理平台 | Vue 3 + Element Plus |
| **官网/会员中心** | `website/` | 产品营销 + 租户自助 | Vue 3 + Vite |
| **移动APP** | `crmAPP/` | 电销外呼工作手机 | uni-app + Vue 3 |
| **企微H5** | `h5/` | 企微工作台应用 | Vue 3 + Vant 4 |
| **微信小程序** | `miniprogram/` | 客户资料收集 | 原生WXML/JS |
| **企微专区程序** | `wecom-program/` | 会话存档与AI分析 | Python + Docker |

## 七、权限与安全体系

### 7.1 角色体系

| 角色 | 数据范围 | 说明 |
|------|----------|------|
| super_admin | all | 超级管理员（所有数据+系统配置） |
| admin | all | 管理员（业务管理） |
| department_manager | department | 部门经理（本部门数据） |
| sales_staff | self | 销售员（个人数据） |
| customer_service | self | 客服（服务相关） |

### 7.2 权限粒度

- **菜单级**：路由meta + 侧边栏权限过滤
- **按钮级**：`v-permission` 指令 + `permissions` 数组
- **数据级**：`dataScope`（all/department/self）
- **字段级**：`SensitiveInfoPermission`（手机号脱敏等）
- **模块级**：`module_status` 开关 + License控制

### 7.3 安全措施

- JWT Token认证 + 会话管理
- Helmet安全头 + CORS配置
- express-rate-limit API限流
- API Key认证（第三方接入）
- License过期只读保护
- 租户容量/席位/存储限制
- 操作审计日志
