# 第一阶段：初始设置篇

> **适用角色**：超级管理员 / 管理员（首次登录必做）  
> **预计耗时**：15-20 分钟  
> **前置条件**：已在官网注册账号，获得了 CRM 系统登录地址和账号密码

---

## 1.1 登录系统

**第 1 步**：打开浏览器（推荐 Chrome 或 Edge），在地址栏输入 CRM 系统地址（注册成功后提供给您）。

**第 2 步**：进入登录页面，看到左侧品牌展示区（云客 CRM 标志），右侧为登录表单：

| 输入框 | 说明 |
|--------|------|
| **用户名** | 输入您的登录账号（注册时设置） |
| **密码** | 输入登录密码 |

**第 3 步**：输入后点击「**登录**」按钮，系统自动跳转到「数据看板」页面。

> ⚠️ 连续输错密码超过 5 次，账户自动锁定 30 分钟。

---

## 1.2 修改密码

**第 1 步**：登录后，点击页面**右上角**您的用户名或头像。

**第 2 步**：在下拉菜单中点击「**修改密码**」。

**第 3 步**：依次输入：

| 输入框 | 操作 |
|--------|------|
| 当前密码 | 输入现在密码 |
| 新密码 | 输入新密码（至少8位，建议字母+数字+符号） |
| 确认新密码 | 再输一次确保一致 |

**第 4 步**：系统显示密码强度（弱/中/强），建议设**强**等级。点击「**修改密码**」。

## 1.4 创建部门

> **为什么先建部门？** 部门和人员是系统骨架，有了它才能归属员工、按部门统计业绩。

**第 1 步**：左侧菜单 → 「**系统管理**」→「**部门管理**」。

**第 2 步**：页面顶部显示四个统计卡片（总部门数/活跃部门/总成员数/层级类型）。

**第 3 步**：点击右上角「**新建部门**」按钮。

**第 4 步**：填写：

| 字段 | 操作 | 必填 |
|------|------|------|
| 部门名称 | 输入如「销售一部」 | ✅ |
| 部门编码 | 英文/数字，如 sales01 | 选填 |
| 上级部门 | 子部门选上级，顶级留空 | 选填 |
| 部门经理 | 可从已有用户中选择 | 选填 |
| 排序 | 数字越小越靠前 | 选填 |
| 状态 | 启用/禁用 | 默认启用 |

**第 5 步**：点击「确定」。重复创建完整结构。

**典型结构**：
```
公司（顶级）
├── 销售一部
│   ├── 销售一组
│   └── 销售二组
├── 销售二部
└── 客服部
```

## 1.5 添加成员（创建用户）

**第 1 步**：左侧菜单 →「**系统管理**」→「**用户管理**」。

**第 2 步**：点击右上角「**新建用户**」。

**第 3 步**：填写：

| 字段 | 操作 | 必填 |
|------|------|------|
| 用户名 | 登录账号，如 zhangsan | ✅ |
| 密码 | 初始密码 | ✅ |
| 姓名 | 真实姓名，如 张三 | ✅ |
| 手机号 | 员工手机号 | ✅ |
| 角色 | 选择角色（见1.6节） | ✅ |
| 部门 | 选择归属部门 | ✅ |
| 邮箱 | 员工邮箱 | 选填 |
| 工号 | 企业工号 | 选填 |
| 职位 | 如 销售顾问 | 选填 |

**第 4 步**：点击「确定」。重复添加所有成员。

## 1.6 角色权限说明

| 角色 | 能看到什么 | 典型分配对象 |
|------|-----------|-------------|
| **超级管理员** | 全部功能和数据 | 老板/IT负责人 |
| **管理员** | 几乎全部，可管理部门和用户 | 销售总监 |

| **销售人员** | 只看自己的客户和订单 | 一线销售 |
| **客服** | 订单审核/物流/售后 | 客服人员 |
| **财务** | 财务数据 | 财务人员 |

> 💡 普通销售看不到「商品管理」「系统管理」是正常的权限控制。

# 第二阶段：商品设置篇

> **适用角色**：管理员 / 超级管理员  
> **前置条件**：已完成第一阶段

---

## 2.1 新增商品分类

> 商品分类是商品的归属，先建分类再加商品。支持两级分类。

**第 1 步**：左侧菜单 →「**商品管理**」→「**商品分类**」。

**第 2 步**：页面以**树形表格**展示分类，点右上角「**新增分类**」。

**第 3 步**：填写：

| 字段 | 操作 |
|------|------|
| 分类名称 | 如「保健食品」（必填） |
| 分类编码 | 如 health（选填） |
| 上级分类 | 子分类才选，顶级留空 |
| 排序 | 数字越小越靠前 |

**第 4 步**：点击「确定」。重复创建完整结构。

**典型分类**：
```
全部商品
├── 保健食品
│   ├── 蛋白粉
│   ├── 维生素
│   └── 鱼油
├── 美妆护肤
└── 虚拟商品
    ├── 充值卡密
    └── 课程资源
```

> ⚠️ 已包含商品的分类不可删除。

---

## 2.2 新增商品

**第 1 步**：左侧菜单 →「**商品管理**」→「**新增商品**」。

**第 2 步**：选择商品类型（⚠️ 保存后不可更改）：

| 类型 | 说明 | 下单后 |
|------|------|--------|
| 📦 普通商品 | 实物，需物流发货 | 需填快递单号 |
| ✨ 虚拟商品 | 无需物流 | 自动发卡密/链接 |

**第 3 步**：填写基本信息：

| 字段 | 操作 | 必填 |
|------|------|------|
| 商品名称 | 如「乳清蛋白粉 5磅」 | ✅ |
| 商品编码 | 手动输入或点「生成」 | ✅ |
| 商品分类 | 级联选择器，选之前建的分类 | ✅ |
| 商品品牌 | 品牌名 | 选填 |
| 商品规格 | 如「5磅/桶」 | 选填 |
| 商品单位 | 件/台/个/盒/包/瓶/袋/套 | 选填 |

**第 4 步**：填写价格库存（右侧）：

| 字段 | 说明 |
|------|------|
| 销售价格 | 售价，如 299.00（必填） |
| 成本价格 | 成本价（选填） |
| 库存数量 | 当前库存（普通商品） |
| 最低库存 | 低于此触发预警 |

**第 5 步**：上传商品图片（支持多张），填写商品描述（富文本编辑器）。

**第 6 步**：点击「**保存**」。

### 虚拟商品额外配置

选择虚拟商品后，需选「虚拟发货方式」：

| 方式 | 说明 | 举例 |
|------|------|------|
| 无需发货 | 无交付物 | 咨询、预约 |
| 卡密发货 | 发放卡密 | 充值卡、激活码 |
| 资源链接 | 发放下载链接 | 课程视频、电子书 |

> 建好后需去「卡密库存」或「资源库存」导入内容（见2.5节）。

---

## 2.3 商品列表管理

**第 1 步**：左侧菜单 →「**商品管理**」→「**商品列表**」。

**第 2 步**：页面顶部三张统计卡片（总商品数/库存预警/缺货商品）。

**第 3 步**：列表操作：

| 操作 | 方法 |
|------|------|
| 搜索 | 按名称/编码搜索 |
| 筛选 | 按分类/状态筛选 |
| 编辑 | 点「编辑」按钮 |
| 上下架 | 切换状态开关 |
| 删除 | 谨慎操作 |

**第 4 步**：右侧快捷按钮：分类管理、批量导入、导出数据、添加商品。

---

## 2.4 库存管理

> 仅对普通商品有效。

**第 1 步**：左侧菜单 →「**商品管理**」→「**库存管理**」。

**第 2 步**：顶部四张卡片（库存总值/商品总数/库存预警/缺货商品）。

**第 3 步**：库存调整：点「**库存调整**」→ 选商品 → 输入数量（正=入库，负=出库）→ 填原因 → 确认。

**第 4 步**：批量导入/导出数据。

---

## 2.5 虚拟商品管理（卡密/资源）

### 卡密库存

**第 1 步**：左侧菜单 →「**商品管理**」→「**卡密库存**」。

**第 2 步**：选对应虚拟商品 → 点「批量导入」→ 上传卡密文件。

> 卡密文件格式（CSV）：首行 `卡号,密码`，每行一条。

### 资源库存

**第 1 步**：左侧菜单 →「**商品管理**」→「**资源库存**」。

**第 2 步**：选对应商品 → 设置资源名称、下载链接、提取码、有效期。

---

## 2.6 商品分析

左侧菜单 →「**商品管理**」→「**商品分析**」。查看销量排行、分类占比、销售趋势。

## 3.1 新增客户

> 这是销售日常最常用的操作：录入新客户。

### 方式一：手动新建

**第 1 步**：左侧菜单 →「**客户管理**」→「**新增客户**」。

**第 2 步**：进入新增客户表单，分多个区域填写：

#### 📋 基本信息（必填）

| 字段 | 操作 | 必填 |
|------|------|------|
| **客户姓名** | 输入客户真实姓名 | ✅ |
| **手机号** | 输入手机号 | ✅ |
| **其他手机号** | 可添加多个备用号码 | 选填 |
| **微信号** | 客户微信号 | 选填 |
| **QQ号** | 客户QQ | 选填 |
| **邮箱** | 客户邮箱 | 选填 |

#### 👤 个人信息

| 字段 | 操作 |
|------|------|
| **性别** | 男/女/未知 |
| **年龄** | 输入年龄 |
| **生日** | 选择日期 |
| **身高(cm)** | 输入身高 |
| **体重(kg)** | 输入体重 |
| **身份证号** | 输入身份证（加密存储） |

#### 📍 地址信息

| 字段 | 操作 |
|------|------|
| **省/市/区** | 级联选择器，三级联动 |
| **街道** | 输入街道名 |
| **详细地址** | 输入门牌号等 |
| **境外地址** | 海外客户使用 |

#### 🏢 企业信息

| 字段 | 操作 |
|------|------|
| **公司名称** | 输入公司名 |
| **行业** | 输入行业 |

#### 📊 业务属性

| 字段 | 操作 | 说明 |
|------|------|------|
| **客户来源** | 下拉选择 | 广告/转介绍/展会/企微/其他 |
| **客户等级** | 下拉选择 | 普通/重要/VIP |
| **跟进状态** | 下拉选择 | 待跟进/跟进中/已成交/已流失 |
| **进粉时间** | 选择日期 | 客户首次接触时间 |
| **销售员** | 选择归属销售 | 默认当前用户 |
| **备注** | 自由输入 | 补充说明 |

#### 🎯 改善目标（行业特色）

可多选预设目标：如「塑形」「增肌」「减脂」等，也支持自定义输入。可填写**疾病史**参考信息。

#### 📝 自定义字段

管理员在系统设置中添加的自定义字段会在此处显示，按需填写。

**第 3 步**：填写完毕后，点击「**保存**」按钮。客户创建成功，自动跳转到客户详情页。

---

### 方式二：Excel批量导入

> 适用于将历史客户数据批量导入系统。

**第 1 步**：左侧菜单 →「**客户管理**」→「**客户列表**」。

**第 2 步**：点击页面上方或操作区的「**导入**」/「**批量导入**」按钮。

**第 3 步**：在批量导入对话框：
1. 下载系统提供的 **Excel 导入模板**
2. 按模板格式填入客户数据
3. 上传文件，系统自动校验数据
4. 预览数据，确认无误
5. 点击「**确认导入**」

**第 4 步**：系统自动检测重复客户（按手机号去重），显示导入结果。

### 方式三：企微自动同步

如果已绑定企业微信，添加企微好友后客户自动创建到 CRM 中（详见第九阶段企微管理篇）。

# 第四阶段：订单管理篇

> **适用角色**：所有有订单权限的用户（销售/经理/客服/管理员）  
> **前置条件**：已完成第一阶段（部门+成员）、第二阶段（商品）、第三阶段（客户）

## 4.1 创建订单（下单）

> 这是销售**最核心的日常操作**：为客户下单。

### 操作步骤

**第 1 步**：左侧菜单 →「**订单管理**」→「**新增订单**」。

**第 2 步**：第一步 —— **选择客户**。

| 操作 | 说明 |
|------|------|
| 搜索框输入 | 输入客户姓名或手机号搜索 |
| 下拉选择 | 从搜索结果中选择客户 |
| 自动带入 | 选中后，客户的收货地址、电话自动填入订单 |

> 💡 如果客户还没有录入系统，可点击「**新建客户**」快速添加。

**第 3 步**：第二步 —— **选择商品**。

| 操作 | 说明 |
|------|------|
| 选择商品 | 从商品库中点击选择（支持搜索） |
| 修改数量 | 调整购买数量 |
| 修改单价 | 可修改销售单价（受权限控制） |
| 自动计算 | 系统自动计算小计和总金额 |

支持一个订单包含多个商品。

**第 4 步**：第三步 —— **填写订单信息**。

#### 📋 客户收货信息

| 字段 | 说明 |
|------|------|
| 收货人 | 默认客户姓名，可修改 |
| 收货电话 | 默认客户手机号，可修改 |
| 收货地址 | 默认客户地址，可修改 |

#### 💰 金额信息

| 字段 | 说明 |
|------|------|
| **总金额** | 商品合计，自动计算 |
| **定金金额** | 如有定金，填写金额 |
| **定金截图** | 上传定金支付凭证（支持多张） |
| **优惠金额** | 优惠/折扣金额 |
| **实付金额** | 总金额 - 优惠金额，自动计算 |

#### 💳 支付信息

| 字段 | 说明 |
|------|------|
| **支付方式** | 微信/支付宝/银行转账/现金/其他 |
| **支付状态** | 未支付/部分支付/已支付/已退款 |

#### 📦 物流信息

| 字段 | 说明 |
|------|------|
| 快递公司 | 选择快递公司（可发货时再填） |
| 快递单号 | 输入运单号（可发货时再填） |
| 发货时间 | 发货日期（可发货时再填） |

#### 🏷️ 其他信息

| 字段 | 说明 |
|------|------|
| 客服微信 | 记录客服微信号，方便联系 |
| 订单来源 | 选择来源渠道 |
| 订单标记 | 标记订单类型 |
| 备注 | 订单备注（富文本） |

**第 5 步**：确认信息无误后，点击「**提交订单**」。

**第 6 步**：提交后，订单会根据系统配置进入不同状态：
- 如果开启了**订单审核** → 订单进入「**待审核**」
- 如果未开启审核 → 订单直接进入「**待发货**」

> 💡 订单提交后会自动生成**唯一订单号**，可在订单列表追踪。

## 4.3 订单审核

> 仅管理员、客服等有审核权限的用户可见。如需开启审核功能，在系统设置中配置。

### 4.3.1 进入审核页面

**第 1 步**：左侧菜单 →「**订单管理**」→「**订单审核**」。

**第 2 步**：页面以审核工作台形式展示：

| 区域 | 内容 |
|------|------|
| **汇总卡片** | 待审核数量、待审核金额、今日新增、超时订单 |
| **快捷筛选** | 一键切换：待审核/紧急/大额订单 |
| **四个Tab** | 待审核 / 已通过 / 已拒绝 / 已取消 |

### 4.3.2 审核操作

**第 1 步**：在「待审核」Tab中，点击某条订单的「**审核**」按钮。

**第 2 步**：查看订单完整信息：
- 客户信息、商品明细、金额
- 定金截图（如有）
- 订单备注

**第 3 步**：做出审核决定：

| 操作 | 后续流程 |
|------|---------|
| **审核通过** | 订单进入「**待发货**」状态，通知发货人员 |
| **审核拒绝** | 订单进入「**审核拒绝**」状态，退回给创建人 |

**第 4 步**：填写审核意见（通过/拒绝时均建议填写）。

**第 5 步**：点击「**确认**」。

### 4.3.3 批量审核

勾选多个订单左侧复选框 → 点击「**批量审核**」→ 选择通过/拒绝 → 填写统一意见 → 确认。一次操作可处理多条订单。

## 4.5 订单取消与退款

### 取消订单

**第 1 步**：在订单列表或详情页，点击「**取消**」按钮。

**第 2 步**：选择取消原因，填写备注说明。

**第 3 步**：提交取消申请。

根据系统配置，取消可能需要审核：
- 无需审核 → 直接取消
- 需要审核 → 进入审核流程

### 退款处理

如订单已付款，取消时需处理退款：

| 字段 | 操作 |
|------|------|
| 退款金额 | 输入退款金额 |
| 退款原因 | 说明退款原因 |
| 退款时间 | 记录退款时间 |

## 6.1 COD代收管理（货到付款）

### COD完整流程

```
1. 创建订单时设置COD金额（如 ¥299）
        ↓
2. 审核通过 → 发货
        ↓
3. 快递员送货 → 代收 ¥299
        ↓
4. 客户签收 → 快递公司定期结算给您
        ↓
5. 财务操作：查看代收 → 操作返款
```

### 查看代收列表

| 列 | 说明 |
|----|------|

| 客户 | 客户姓名 |
| COD金额 | 代收金额 |
| 代收状态 | 待返款/已返款/已取消 |
| 返款金额 | 已返款金额 |
| 返款时间 | 返款日期 |
| 创建时间 | 订单创建日期 |

**第 1 步**：在代收列表找到待返款订单，点击「**返款**」。

| 备注 | 补充说明 |

**第 3 步**：点击确认。订单标记为「已返款」。

### 改代收

如需修改代收金额：点击「**改代收**」→ 输入新金额 → 确认。

### 取消代收

**第 1 步**：左侧菜单 →「**订单管理**」→「**取消代收申请**」（销售发起）。

# 第八阶段：资料管理篇

> **适用角色**：管理员/部门经理/客服（资料列表+分配）、销售（客户查询）、管理员（回收站）  
> **前置条件**：有已完成签收的订单（签收后资料自动进入列表）

---

## 8.1 资料管理概述

> 客户订单签收后，该客户资料自动进入「资料库」。管理员可将资料分配给销售跟进后续复购或售后关怀。

**资料从哪来**：
- 订单签收后**自动转入**，状态为「待分配」
- 管理员/经理可**手动新增**

**资料管理四个状态**：

| 状态 | 说明 |
|------|------|
| 待分配 | 刚进入资料库，还未分配给任何人 |
| 已分配 | 已分配给某个销售跟进 |
| 已封存 | 暂时不需要跟进，封存起来 |
| 已回收 | 从回收站恢复了但尚未重新分配 |

---

## 8.2 资料列表与分配

### 8.2.1 进入资料列表

**第 1 步**：左侧菜单 →「**资料管理**」→「**资料列表**」。

**第 2 步**：页面顶部四张卡片：

| 卡片 | 含义 |
|------|------|
| 资料总数 | 所有资料总量 |
| 待分配 | 尚未分配的数量 |
| 已分配 | 已分配的数量 |
| 订单总额 | 所有资料订单金额合计 |

**第 3 步**：快速筛选：今天/昨天/本周/最近30天/本月/今年/全部。

**第 4 步**：列表展示每条资料的详细信息：

| 列 | 说明 |
|----|------|
| 客户姓名 | 脱敏显示 |
| 手机号 | 脱敏显示 |
| 订单号 | 关联的订单 |
| 订单金额 | 该笔订单金额 |
| 状态 | 待分配/已分配/已封存/已回收 |
| 归属人 | 分配给谁 |
| 分配时间 | 操作时间 |

### 8.2.2 分配资料

> 核心操作：将资料分配给销售跟进。

**第 1 步**：勾选一条或多条「待分配」资料。

**第 2 步**：点击「**分配**」按钮，选择分配方式：

| 方式 | 说明 | 适用场景 |
|------|------|---------|
| **指定分配** | 手动选人 | 灵活分配 |
| **轮流分配** | 部门成员轮流分 | 公平分配 |
| **随机分配** | 随机分配 | 快速分配大量资料 |
| **经理分配** | 经理统一分配 | 经理管控 |

**第 3 步**：选择目标销售 → 填写备注 → 确认分配。

**第 4 步**：该销售客户列表中会出现该客户，可开始跟进。

### 8.2.3 分配统计

列表页面可查看每个销售的分配统计：今日/本周/本月分配数、平均订单金额。

## 8.3 资料封存

**第 2 步**：设置：

| 字段 | 操作 |

| 封存时长 | 选择天数 |
| 解封时间 | 自动解封时间 |

| 手机号 | 精确匹配 |
| 客户编码 | 按编码查找 |
| 订单号 | 查找订单对应客户 |

| 群成员数 | 当前人数 |
| 创建时间 | 建群时间 |

**第 3 步**：群运营功能：
- **群发消息**：选择群→编辑消息→批量发送
- **入群欢迎语**：设置新人入群自动欢迎
- **群统计**：查看活跃度、人数变化趋势

## 9.6 会话存档

> 合规功能：存档企业与客户的聊天记录，满足金融、医疗等行业的监管要求。

**第 1 步**：左侧菜单 →「**企微管理**」→「**会话存档**」。

**第 2 步**：开启存档功能（需先在企微管理后台开通会话存档权限）。

**第 3 步**：配置存档生效成员（选择哪些成员的聊天需要存档）。

**第 4 步**：查看聊天记录：
- 按成员/客户/时间筛选
- **全文搜索**：输入关键词搜索所有聊天内容
- **会话时间线**：按时间轴查看完整对话
- **聊天详情**：文字、图片、文件等

> ⚠️ 存档会告知双方「聊天内容已存档」，请合规使用。

## 9.7 侧边栏应用

> 在企微聊天窗口右侧打开CRM面板，边聊边查客户信息，不用切换系统。

**第 2 步**：配置侧边栏应用：

| 配置项 | 说明 |
|--------|------|

| 应用类型 | 客户详情/订单查询/知识库/话术/自定义 |
| 应用地址 | 自动生成 |
| 可见范围 | 哪些成员可用 |

**第 3 步**：保存后在企微聊天窗口右侧即可打开。

**侧边栏主要功能**：
- **客户详情**：查看当前聊天客户的CRM完整信息
- **客户画像**：消费数据/标签/评分/订单趋势
- **快捷下单**：在侧边栏直接为客户创建订单
- **话术库**：快捷回复常用话术
- **资料收集**：发送小程序卡片让客户自助填写资料

## 10.2 部门成员权限补充设置

> 在前面第一阶段已建好部门和成员，这里补充一些高级操作。

### 部门详情

**第 2 步**：进入部门详情页，可进行：

**第 1 步**：左侧菜单 →「**系统管理**」→「**超管面板**」（仅超级管理员可见）。

## 10.3 角色权限管理

**第 1 步**：左侧菜单 →「**系统管理**」→「**角色权限**」。

| 配置项 | 说明 |
|--------|------|

## 10.4 消息与公告管理

## 10.5 安全设置

**第 1 步**：左侧菜单 →「**系统管理**」→「**系统设置**」→ 切换到「**安全设置**」Tab。

**第 2 步**：可配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 密码最小长度 | 用户密码最少位数 | 8位 |
| 密码过期天数 | 强制修改周期 | 90天 |
| 登录失败锁定 | 多少次错误后锁定 | 5次 |
| 锁定时间 | 锁定多久 | 30分钟 |
| 会话超时 | 多久无操作自动退出 | 120分钟 |
| IP白名单 | 限制登录IP | 空（不限制） |

## 10.6 存储与文件设置

**第 1 步**：左侧菜单 →「**系统管理**」→「**系统设置**」→ 切换到「**存储设置**」Tab。

| 配置项 | 说明 |
|--------|------|

| 允许的文件类型 | jpg/png/pdf/doc等 |
| 图片压缩 | 是否启用压缩 |
| 压缩质量 | 低/中/高 |

## 10.7 短信/通话/邮件设置

### 短信设置

**第 1 步**：切换到「**短信设置**」Tab。

### 邮件设置

切换到「**邮件设置**」Tab，配置SMTP：服务器、端口、发件邮箱、密码。
