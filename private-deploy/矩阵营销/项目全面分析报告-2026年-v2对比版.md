| 管理后台配置 | 无 | **`MobileAppTab.vue`** 完整配置面板 |

| 版本管理 | 无 | 多版本上传、启用/禁用、下载计数 |

### APP 下载全链路

```
管理后台(MobileAppTab.vue) → 上传 APK/IPA → mobile_app_packages 表
                                    ↓
CRM前端(/mobile-app-download) → GET /mobile-app/list → 展示下载卡片
    ├── Android → 点击下载 → GET /mobile-app/download/:id → 下载计数+1
    └── iOS    → 外部链接 → 跳转 App Store / 蒲公英 / 企业签名链接
```

### APP构建方式

APP 源码位于 `crmAPP/` 目录（gitignore 豁免，独立构建系统），使用 HBuilderX / uniapp 打包生成原生 APK 和 IPA 安装包。管理后台可以上传安装包或配置外部下载链接（如 App Store、蒲公英分发平台）。
