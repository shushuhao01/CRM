## 一、项目全景（更新）

### 1.1 产品定位

云客CRM是一个 **SaaS + 私有部署双模式** 的完整CRM商业产品体系，包含5个子系统：

| 子系统 | 技术栈 | 代码量（最新统计） | 文件数 | 端口 | 定位 |
|--------|--------|--------|--------|------|------|
| **Backend** | Node.js + Express + TypeORM + MySQL | **78,043行** | **252文件** | 3000 | 统一后端API服务 |
| **CRM前端** | Vue 3 + TS + Element Plus + Pinia | **242个Vue + 146个TS** | **388文件** | 5173 | 主业务系统（员工使用） |
| **Admin后台** | Vue 3 + TS + Element Plus + Axios | **20,683行**（33Vue+9TS） | **42文件** | 5174 | 平台管理后台（运营方使用） |
| **Website官网** | Vue 3 + TS + 纯手写SCSS | **8,267行**（15Vue+4TS+2SCSS） | **21文件** | 8080 | 产品展示 + 注册 + 支付 |
| **crmAPP** | uni-app + Vue 3 + TS | **10,106行**（18Vue+14TS） | **32文件** | — | 电销外呼手机APP |

**📊 全项目代码总量：约 330,000+ 行（含地址数据文件约55,000行，较3月22日增长约20,000行）**

### 1.3 商业闭环

```
潜在客户访问官网 → 选择套餐 → 注册/支付 → 自动开通租户 → 下发登录信息
                                                                     ↓
管理后台运营管理 ← 授权/续费/模块控制 ← 管理员 ← 分配账号 ← 登录CRM系统
                                                              ↓
                                                     CRM日常使用（客户/订单/物流/业绩）
                                                              ↓
                                                     移动端APP → 外呼/录音/跟进
```

## 二、总体架构

### 2.1 系统架构图

```
                    ┌────────────────────┐
                    │    Website 官网     │ ← 产品展示、注册、支付
                    │    (Vue 3 + SCSS)  │
                    │    Port: 8080      │
                    └────────┬───────────┘
                             │ /api/v1/public/*
┌────────────────┐   ┌──────┴───────────┐   ┌────────────────┐
│  CRM 主前端    │──→│   Backend API    │←──│  Admin 后台    │
│  (Vue 3 + EP)  │   │   (Express.js)   │   │  (Vue 3 + EP)  │
│  Port: 5173    │   │   Port: 3000     │   │  Port: 5174    │
└────────────────┘   │                   │   └────────────────┘
                     │  REST API         │
┌────────────────┐   │  + WebSocket      │   ┌────────────────┐
│   crmAPP       │←─→│  + Socket.IO     │   │    MySQL 8.0   │
│  (uni-app)     │WS │  + node-cron     │──→│  (共享数据库)   │
│  Android/iOS   │   │  + PM2           │   └────────────────┘
└────────────────┘   └──────────────────┘
```

### 2.2 部署模式

| 模式 | 环境变量 | 说明 |
|------|---------|------|
| **私有部署** | `DEPLOY_MODE=private` | 单租户，授权码激活，数据tenant_id=NULL |
| **SaaS云端** | `DEPLOY_MODE=saas` | 多租户，共享数据库，tenant_id隔离 |

### 2.3 通信方式

| 连接类型 | 技术 | 用途 |
|---------|------|------|
| HTTP REST | Express + JWT | 所有CRUD操作 |
| WebSocket (Socket.IO) | socket.io | PC端实时消息推送 |
| WebSocket (ws) | ws | 移动端通话指令/状态同步 |
| 定时任务 | node-cron | 授权过期检查、支付提醒、消息清理、超时提醒、业绩报表 |
