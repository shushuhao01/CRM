# 云客APP外呼助手 - 开发启动提示词

> 复制以下内容发送给 Claude Opus 4.5 开始开发

---

## 直接复制使用的提示词

```
你好！我需要你作为一名资深移动端开发工程师，帮我开发「云客APP外呼助手」。

## 项目简介

这是一款配合 PC 端 CRM 系统使用的电销工作手机 APP，核心功能：
- PC 端可以远程控制手机拨号
- 手机端可以主动外呼、接听来电
- 通话录音自动上传服务器
- 通话数据实时同步
- 支持多租户部署（不同公司使用不同服务器）

## 技术栈

- uni-app 3.x + Vue 3 + TypeScript
- Pinia 状态管理
- socket.io-client WebSocket 通信
- 目标平台：Android 7.0+ / iOS 12.0+

## 核心文档

请先阅读以下三个文档，它们包含了完整的产品设计、接口定义和技术实现细节：

1. **docs/APP产品设计文档.md** - 产品定位、页面原型、UI规范、交互设计
2. **docs/APP接口文档.md** - 所有 API 接口定义、WebSocket 协议
3. **docs/APP开发技术文档.md** - 技术架构、代码示例、原生功能集成

## 开发优先级

**P0 必须完成**：
1. 服务器配置页（多租户支持）
2. 登录页
3. 扫码绑定设备
4. 首页（连接状态、今日概览）
5. WebSocket 连接保持
6. 接收 PC 端拨号指令并执行
7. 通话状态上报
8. 通话结束处理（上传录音）

**P1 重要功能**：
1. 通话记录列表
2. 拨号盘
3. 通话跟进（备注、标签）
4. 统计页面
5. 设置页

## UI 设计规范

- 主色：#34D399（翠绿色）
- 渐变：linear-gradient(135deg, #6EE7B7 0%, #34D399 100%)
- 圆角：按钮 8px、卡片 12px
- 页面边距：16px

## 开发要求

1. 使用 Composition API + setup 语法糖
2. TypeScript 类型定义完整
3. 组件样式使用 SCSS + scoped
4. 代码注释清晰
5. 错误处理完善

## 开始任务

请先阅读上述三个文档，然后告诉我：
1. 你对项目的整体理解
2. 你计划的开发顺序
3. 可能遇到的技术难点
4. 需要我补充的信息

准备好后，我们从「服务器配置页」开始开发。
```
