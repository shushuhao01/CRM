## 二、Register.vue 三步注册流程

### Step 1 - 选择套餐

- SaaS版：免费试用 / 基础版 / 专业版 / 企业版
- 私有部署版：标准版 / 专业版 / 企业版
- SaaS非免费：月付/年付切换 + 订阅自动续费勾选（默认勾选）
- 私有部署：年度授权/永久买断切换

### Step 2 - 填写信息

- 企业名称、联系人、手机号、验证码、邮箱(可选)、密码(可选)
- **免费试用专属**：到期自动续费勾选 + 选择续费套餐
- 同意服务协议

### 提交注册后的分支逻辑

```
                    注册 API 调用
                        │
                   注册成功？(code===0)
                   ╱          ╲
                 是              否 ──> 提示错误
                 │
         选的是 FREE_TRIAL？
          ╱              ╲
        是                 否（付费套餐）
        │                  │
  勾选了自动续费？     勾选了订阅续费？(subscriptionChecked)
   ╱          ╲        ╱          ╲
  是           否      是           否
  │            │      │            │
  v            v      v            v
签约模式    直接跳转   签约模式     普通支付模式
(Step3)   成功页     (Step3)     (Step3)
```

## 三、Step 3 - 核心分支

### 路径总览

| # | 场景 | 路径 | 步骤数 |
|---|------|------|--------|
| 1 | 免费试用 - 不勾选续费 | 选套餐 → 填信息 → 成功页 | 3步 |
| 2 | 免费试用 - 勾选续费（不可跳过） | 选套餐 → 填信息 → 扫码签约 → 成功页 | 4步 |
| 3 | 付费套餐 - 签约成功 | 选套餐 → 填信息 → 扫码签约 → 成功页 | 4步 |
| 4 | 付费套餐 - 签约取消 → 正常付费成功 | 选套餐 → 填信息 → 签约取消 → 扫码付费 → 成功页 | 5步 |
| 5 | 付费套餐 - 签约取消 → 付费超时 | 选套餐 → 填信息 → 签约取消 → 3分钟超时 → 会员中心 | 5步 |
| 6 | 付费套餐 - 不勾选订阅 → 普通支付 | 选套餐 → 填信息 → 正常支付/对公转账 → 成功页 | 4步 |
| 7-8 | 私有部署 | 同路径6，不支持订阅签约 | 4步 |

### 模式A：签约模式（免费试用+勾选续费 或 付费套餐+勾选订阅）

```
┌────────────────────────────────────────────────────────────────────┐
│                    签约模式 (signingMode = true)                    │
│                                                                    │
│  ┌─ 注册成功提示 ─────────────────────────────────────────┐        │
│  │ ✅ 账号已创建成功！                                     │        │
│  │ 租户编码: xxx    授权码: xxx    管理员: xxx             │        │
│  └─────────────────────────────────────────────────────────┘        │
│                                                                    │
│  signingStatus 状态机：                                             │
│                                                                    │
│  ┌────────┐  开始签约  ┌─────────┐  签约成功  ┌──────────┐        │
│  │  idle  │─────────>│ waiting │─────────>│ success  │        │
│  │选择渠道 │           │等待扫码  │           │签约完成   │        │
│  └────────┘           └────┬────┘           └────┬─────┘        │
│                            │                      │               │
│                    ┌───────┴──────┐          开始使用              │
│                    │ 免费试用？    │               │               │
│                    ╱            ╲               v               │
│                  是              否         /pay-success         │
│                  │         (付费套餐)                              │
│            无取消按钮    [取消签约]                                 │
│            只能完成签约       │                                    │
│                         v                                        │
│                    ┌────────┐  确认支付  ┌────────────┐           │
│                    │ paying │─────────>│pay-waiting │           │
│                    │⚠未完成  │           │3分钟倒计时  │           │
│                    │选支付方式│           │显示二维码   │           │
│                    └────────┘           └─────┬──────┘           │
│                                        ╱           ╲             │
│                                  支付成功         3分钟超时        │
│                                    │              或取消          │
│                                    v               v             │
│                              ┌────────────┐  ┌──────────────┐    │
│                              │pay-success │  │  会员中心     │    │
│                              │付费成功     │  │ (未付费状态) │    │
│                              └─────┬──────┘  └──────────────┘    │
│                                    │                              │
│                                    v                              │
│                              /pay-success                         │
└────────────────────────────────────────────────────────────────────┘
```

**关键规则：**
- **免费试用**签约：`waiting`状态**没有取消按钮**，必须完成签约
- **付费套餐**签约：`waiting`状态有"取消签约"按钮，取消后进入正常付费
- 正常付费倒计时**3分钟**，超时自动跳转会员中心（未付费状态）
- 取消支付也直接跳转会员中心

## 四、会员中心 - 未付费用户

未完成支付的用户跳转到会员中心后：

| 状态 | 会员中心显示 | 可执行操作 |
|------|------------|-----------|
| 未付费 | 套餐显示"未订购"，无授权码 | 选择套餐并支付、开通订阅 |
| 未签约 | 订阅状态显示"暂未开通自动续费" | 开通自动续费签约 |
| 已付费已签约 | 正常显示套餐、授权码等全部信息 | 管理订阅、查看账单 |

---

## 五、倒计时规则

| 场景 | 倒计时时长 | 到期行为 |
|------|----------|---------|
| 签约模式下的正常付费(签约取消后) | **3分钟** | 自动跳转会员中心 |
| 普通支付模式(非签约) | 30分钟 | 提示订单过期，回到订单确认页 |

## 六、状态机完整图

```
核心状态变量:
  step: 1 | 2 | 3
  signingMode: boolean
  signingStatus: 'idle' | 'waiting' | 'success' | 'paying' | 'pay-waiting' | 'pay-success'
  isTrialSigning: boolean  (免费试用的签约 vs 付费套餐的签约)
  paymentOrder: null | {...}  (普通支付模式下使用)

完整状态转移:

Step1 ──> Step2 ──> 注册API
  │
  ├──> /pay-success (FREE_TRIAL 无自动续费)
  │
  ├──> Step3 + signingMode (FREE_TRIAL + 自动续费)
  │     └──> idle ──> waiting ──> success ──> /pay-success
  │         (无取消按钮，必须签约完成)
  │
  ├──> Step3 + signingMode (付费套餐 + 勾选订阅)
  │     └──> idle ──> waiting ──> success ──> /pay-success
  │                    │
  │                    └──> [取消签约]
  │                         │
  │                    paying ──> pay-waiting ──> pay-success ──> /pay-success
  │                                   │
  │                               [3分钟超时/取消]
  │                                   v
  │                            /member/dashboard (未付费)
  │
  └──> Step3 + signingMode=false (付费套餐 不勾选订阅 / 私有部署)
        └──> 订单确认 ──> 创建订单 ──> 二维码(30分) ──> /pay-success
                                   └──> 对公转账 ──> /pay-success?payType=bank
```

## 七、关键 API 接口

| 接口 | 方法 | 作用 |
|------|------|------|
| `/api/v1/public/packages` | GET | 获取所有套餐 |
| `/api/v1/public/register/send-code` | POST | 发送手机验证码 |
| `/api/v1/public/register` | POST | 注册(创建租户)，可返回 memberToken |
| `/api/v1/public/payment/create` | POST | 创建支付订单 |
| `/api/v1/public/payment/query/:orderNo` | GET | 查询支付状态(轮询3秒) |
| `/api/v1/public/register/subscribe` | POST | 发起签约 |
| `/api/v1/public/register/subscribe-status/:id` | GET | 查询签约状态(轮询3秒) |
