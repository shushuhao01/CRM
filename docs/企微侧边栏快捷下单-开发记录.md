## 📋 功能概述

在企业微信聊天侧边栏中，员工可以通过快捷下单功能直接为客户创建订单，无需切换到CRM系统。与CRM新增订单保持完全一致的字段配置，根据租户设置动态显示。

## 🔧 技术实现

### 依赖的Store
- `useCustomerFieldConfigStore`：控制客户字段的启用/必填
- `useOrderFieldConfigStore`：控制订单来源选项和订单自定义字段

### API端点
| 功能 | 端点 | 方法 |
|------|------|------|
| 搜索客户 | `/customers` | GET |
| 手机号查重 | `/customers/check-exists` | GET |
| 新建客户 | `/customers` | POST |
| 获取客户详情 | `/customers/:id` | GET |
| 搜索产品 | `/products` | GET |
| 获取快递公司 | `/logistics/companies/active` | GET |
| 获取支付方式 | `/system/payment-methods` | GET |
| 获取订单 | `/orders` | GET |
| 提交订单 | `/orders` | POST |

### 文件变更
- `src/views/Wecom/components/SidebarPreview.vue` — 主要开发文件（~1450行）
- `src/views/Wecom/Sidebar.vue` — 添加 `@switch-app` 事件处理和 `handleSwitchPreviewApp` 方法
