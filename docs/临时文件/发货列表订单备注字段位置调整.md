# 发货列表订单备注字段位置调整

## 修改时间
2026-04-19

## 修改内容

### 1. 调整订单备注字段位置
在物流管理 - 发货列表中，将"订单备注"字段移动到"下单时间"后面，方便发货时查看订单备注信息。

### 2. 添加溢出悬浮提示
为订单号和下单时间字段添加 `showOverflowTooltip` 配置，当内容溢出时鼠标悬浮显示完整信息。

### 具体修改

**文件：** `src/views/Logistics/Shipping.vue`

**修改位置：** `baseTableColumns` 数组配置

**调整内容：**
1. ✅ 将订单备注字段从原位置（支付方式后面）移动到下单时间后面
2. ✅ 调整订单备注列宽从 120px 改为 100px（更窄，节省空间）
3. ✅ 为订单号字段添加 `showOverflowTooltip: true`
4. ✅ 为下单时间字段添加 `showOverflowTooltip: true`
5. ✅ 订单备注保留 `showOverflowTooltip: true` 配置

### 用户体验优化

✅ **发货人员可以更早看到订单备注**
- 订单备注紧跟在下单时间后面，在查看订单时更容易注意到
- 不需要横向滚动到很后面才能看到备注信息

✅ **列宽优化**
- 订单备注宽度从 120px 缩小到 100px，节省屏幕空间
- 溢出内容通过鼠标悬浮提示完整显示，不影响信息查看

✅ **悬浮提示完善**
- 订单号：当订单号过长时，鼠标悬浮显示完整订单号
- 下单时间：当时间格式较长时，鼠标悬浮显示完整时间
- 订单备注：当备注内容过长时，鼠标悬浮显示完整备注

✅ **保持一致性**
- 与其他需要悬浮提示的字段保持一致的交互方式
