# 订单审核列表产品字段添加说明

## 修改时间
2026-04-17

## 需求描述
在CRM系统的订单管理-订单审核列表中，在订单备注字段后面添加产品字段，显示订单对应的商品信息。商品和订单备注字段内容溢出时，鼠标悬浮显示完整内容。

## 实现内容

### 1. 数据结构修改

#### 1.1 AuditOrder接口扩展
在 `src/views/Order/Audit.vue` 中的 `AuditOrder` 接口添加 `products` 字段：

```typescript
interface AuditOrder {
  // ... 其他字段
  products?: Array<{     // 商品列表
    id: string
    name: string
    sku?: string
    price: number
    quantity: number
    image?: string
  }>
  // ... 其他字段
}
```

### 2. 表格列配置修改

#### 2.1 添加产品列
在 `tableColumns` 配置中，在订单备注列后面添加产品列：

```typescript
{
  prop: 'remark',
  label: '订单备注',

},
{
```

### 3. 模板插槽实现

#### 3.1 订单备注列插槽
添加订单备注列的自定义插槽，实现溢出悬浮提示：

```vue
<template #column-remark="{ row }">
  <el-tooltip
    :content="row.remark || '无备注'"
    placement="top"
    :disabled="!row.remark || row.remark.length <= 20"
  >
    <span class="text-overflow">{{ row.remark || '-' }}</span>
  </el-tooltip>
</template>
```

#### 3.2 产品列插槽
添加产品列的自定义插槽，显示商品信息并实现悬浮提示：

```vue
<template #column-products="{ row }">
  <el-tooltip
    placement="top"
    :disabled="!row.products || row.products.length === 0"
  >
    <template #content>
      <div class="products-tooltip">
        <div
          v-for="(product, index) in row.products"
          :key="product.id"
          class="product-item-tooltip"
        >
          <div class="product-name-tooltip">{{ index + 1 }}. {{ product.name }}</div>
          <div class="product-detail-tooltip">
            <span v-if="product.sku">SKU: {{ product.sku }}</span>
            <span>数量: {{ product.quantity }}</span>
            <span>单价: ¥{{ (product.price || 0).toFixed(2) }}</span>
          </div>
        </div>
      </div>
    </template>
    <span class="text-overflow products-text">
      <template v-if="row.products && row.products.length > 0">
        {{ row.products.map((p: any) => `${p.name}×${p.quantity}`).join('、') }}
      </template>
      <template v-else>-</template>
    </span>
  </el-tooltip>
</template>
```

### 4. 数据转换处理

## 功能特点

### 1. 产品列显示
- 显示格式：`商品名称×数量`，多个商品用顿号分隔
- 例如：`iPhone 14 Pro×2、AirPods Pro×1`
- 无商品时显示：`-`

### 2. 悬浮提示内容
产品列悬浮时显示详细信息：
- 商品序号和名称
- SKU编码（如果有）
- 购买数量
- 商品单价

### 3. 订单备注优化
- 内容超过20个字符时，鼠标悬浮显示完整内容
- 无备注时显示：`-`

### 4. 响应式设计
- 列宽自适应，最小宽度180px
- 文本溢出自动省略
- 悬浮提示最大宽度400px
