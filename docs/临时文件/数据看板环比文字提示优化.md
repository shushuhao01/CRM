# 数据看板环比文字提示优化

## 需求描述
在数据看板的汇总卡片中，环比指标后面添加小字提示（如"较昨日"、"较上月"），让用户更清楚环比的对比期。文字要小一点、不明显、不突出。

## 实现方案

### 1. 模板修改
在环比数字后面添加 `trendLabel` 显示：

```vue
<div class="metric-change" :class="metric.trend">
  <el-icon :size="12">
    <ArrowUp v-if="metric.trend === 'up'" />
    <ArrowDown v-if="metric.trend === 'down'" />
    <Minus v-if="metric.trend === 'stable'" />
  </el-icon>
  <span>{{ metric.change }}</span>
  <span class="trend-period">{{ metric.trendLabel }}</span>
</div>
```

### 2. 数据结构修改
在 `metrics` 数组的每个指标中添加 `trendLabel` 字段：

| 指标 | 环比文字 |
|------|---------|
| 今日订单 | 较昨日 |
| 新增客户 | 较昨日 |
| 今日业绩 | 较昨日 |
| 本月单数 | 较上月 |
| 本月业绩 | 较上月 |
| 待处理售后 | 较昨日 |
| 待审核订单 | 较昨日 |
| 待发货订单 | 较昨日 |

### 3. 样式优化
添加 `.trend-period` 样式，让文字小一点、不突出：

```css
.metric-change .trend-period {
  font-size: 10px;
  opacity: 0.6;
  margin-left: 2px;
}
```

特点：
- `font-size: 10px` - 比环比数字（12px）小2px
- `opacity: 0.6` - 透明度60%，不突出
- `margin-left: 2px` - 与环比数字保持小间距

## 视觉效果
```
+12.5% 较昨日
  ↑     ↑
 正常  小字淡色
```

- 个人业绩页面：环比文字根据筛选条件动态变化（较昨日、较上周、较上月等）
- 数据看板：环比文字固定（今日指标显示"较昨日"，本月指标显示"较上月"）
- 两个页面的样式保持一致，都是小字、淡色、不突出
