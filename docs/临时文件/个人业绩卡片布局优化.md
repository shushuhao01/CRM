# 个人业绩卡片布局优化

## 修改时间
2026-02-27

## 修改内容

### 问题描述
个人业绩页面的汇总卡片中,环比指标显示在标题下方,导致卡片高度过高(120px),占用过多垂直空间。

### 优化方案
将环比指标从标题下方移动到标题后面(同一行),并自适应调整卡片高度。

### 修改文件
`src/views/Performance/Personal.vue`

### 1. 模板结构调整

#### 修改前
```vue
<div class="card-info">
  <div class="card-value">{{ performanceData.totalSales }}</div>
  <div class="card-label">总销售额</div>
  <div class="card-trend">
    <span class="trend">...</span>
    <span class="trend-text">较上期</span>
  </div>
</div>
```

#### 修改后
```vue
<div class="card-info">
  <div class="card-header-row">
    <div class="card-label">总销售额</div>
    <div class="card-trend">
      <span class="trend">...</span>
      <span class="trend-text">较上期</span>
    </div>
  </div>
  <div class="card-value">{{ performanceData.totalSales }}</div>
</div>
```

### 2. 样式调整

#### 卡片高度优化
```css
.overview-card {
  height: 90px;  /* 从 120px 减少到 90px */
}
```

#### 新增布局样式
```css
.card-info {
  flex: 1;
  display: flex;
  flex-direction: column;
  gap: 4px;
}

.card-header-row {
  display: flex;
  justify-content: space-between;
  align-items: center;
  margin-bottom: 4px;
}
```

#### 标签样式调整
```css
.card-label {
  font-size: 14px;
  color: #909399;
  /* 移除 margin-bottom: 8px; */
}
```

#### 环比趋势样式调整
```css
.card-trend {
  display: flex;
  align-items: center;
  gap: 4px;  /* 从 8px 减少到 4px */
}
```

#### 数值样式调整
```css
.card-value {
  font-size: 24px;
  font-weight: bold;
  color: #303133;
  /* 移除 margin-bottom: 4px; */
}
```

### 3. 布局效果

#### 修改前
```
┌─────────────────────────┐
│ [图标]  ¥4,632         │
│         总销售额        │
│         ↑ 5% 较上期    │  高度: 120px
└─────────────────────────┘
```

#### 修改后
```
┌─────────────────────────┐
│ [图标]  总销售额 ↑ 5% 较上期 │
│         ¥4,632         │  高度: 90px
└─────────────────────────┘
```

### 优化效果

1. **空间利用**: 卡片高度从120px减少到90px,节省25%的垂直空间
2. **信息密度**: 标题和环比指标在同一行,信息更紧凑
3. **视觉层次**: 标题和趋势在上方,数值在下方,层次更清晰
4. **响应式**: 使用flexbox布局,自适应不同屏幕尺寸

### 应用范围

此优化应用于个人业绩页面的4个汇总卡片:
- 总销售额
- 订单数量
- 签收业绩
- 签收订单数量

### 兼容性

- 保持原有的hover动画效果
- 保持原有的图标颜色和样式
- 保持原有的数据格式化逻辑
- 不影响其他页面和组件

### 测试要点

1. 验证卡片高度是否正确(90px)
2. 验证标题和环比指标是否在同一行
3. 验证数值是否正确显示
4. 验证hover效果是否正常
5. 验证不同屏幕尺寸下的显示效果
6. 验证环比趋势的上升/下降箭头和颜色是否正确
