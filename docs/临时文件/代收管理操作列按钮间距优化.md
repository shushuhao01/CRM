# 代收管理操作列按钮间距优化

## 需求描述
代收管理页面的操作列按钮（详情、改代收、返款）比较拥挤，需要增加按钮之间的间距，让布局更协调。

## 实现方案

### 1. 模板修改
为每个操作按钮添加 `action-btn` class：

```vue
<el-table-column label="操作" width="220" fixed="right">
  <template #default="{ row }">
    <el-button type="primary" link size="small" @click="showDetailDialog(row)" class="action-btn">详情</el-button>
    <template v-if="activeTab !== 'zero'">
      <el-tooltip>
        <span>
          <el-button
            type="warning"
            link
            size="small"
            @click="showCodDialog(row)"
            class="action-btn"
          >改代收</el-button>
        </span>
      </el-tooltip>
      <el-tooltip>
        <span>
          <el-button
            type="success"
            link
            size="small"
            @click="handleReturn(row)"
            class="action-btn"
          >返款</el-button>
        </span>
      </el-tooltip>
    </template>
  </template>
</el-table-column>
```

### 2. 样式添加
添加 `.action-btn` 样式，设置右边距：

```scss
// 操作列按钮间距
.action-btn {
  margin-right: 8px;
  
  &:last-child {
    margin-right: 0;
  }
}
```

## 优化效果

### 优化前
```
详情改代收返款
```
按钮紧挨在一起，视觉上拥挤

### 优化后
```
详情  改代收  返款
```
按钮之间有 8px 的间距，视觉上更协调

## 技术细节

1. **间距设置**：每个按钮右侧 8px 间距
2. **最后一个按钮**：使用 `:last-child` 选择器，最后一个按钮不添加右边距
3. **响应式**：间距在不同屏幕尺寸下保持一致
4. **兼容性**：不影响按钮的禁用状态和 tooltip 功能

- 保持与其他页面操作列的一致性
