## 使用说明

### 批量导出
1. 进入客户管理 -> 客户列表
2. 使用筛选条件筛选需要导出的客户
3. 点击"批量导出"按钮
4. 确认导出提示
5. 等待文件下载完成

### 选中导出
1. 在客户列表中勾选需要导出的客户
2. 点击"导出选中"按钮
3. 确认导出提示
4. 等待文件下载完成
