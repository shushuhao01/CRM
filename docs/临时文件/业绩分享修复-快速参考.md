# 业绩分享报错修复 - 快速参考

## 🔴 问题
开发环境访问业绩分享页面报错：`can't access property "map", share.shareMembers is undefined`

## 🎯 原因
开发环境数据库缺少两个表：
- `performance_shares`
- `performance_share_members`

## ⚡ 快速修复（3步）

### 1️⃣ 打开 Navicat
连接到开发环境数据库 `crm_local`

### 2️⃣ 执行创建脚本
复制并执行：`scripts/create-performance-shares-tables-dev.sql`

### 3️⃣ 刷新页面
清除浏览器缓存，刷新业绩分享页面

## ✅ 验证成功
页面不再报错，可以正常显示业绩分享列表
