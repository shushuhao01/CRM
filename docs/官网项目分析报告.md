# 云客CRM官网项目分析报告

## 项目概述

云客CRM官网是一个基于 Vue 3 + TypeScript 的现代化单页应用，用于产品展示、功能介绍、套餐销售和用户注册。

## 技术栈

### 前端框架
- **Vue 3.4.0** - 使用 Composition API
- **TypeScript 5.3.0** - 类型安全
- **Vite 5.0.0** - 快速构建工具

### 状态管理与路由
- **Vue Router 4.2.5** - 客户端路由
- **Pinia 2.1.7** - 状态管理
- **@vueuse/core 10.7.0** - Vue 组合式工具库

### 样式
- **SCSS/Sass 1.69.0** - CSS 预处理器

## 与后端对接

### 需要的 API 接口

1. **用户注册**
   - `POST /api/v1/website/register`
   - 参数: phone, code, password, packageId

2. **发送验证码**
   - `POST /api/v1/website/send-code`
   - 参数: phone

3. **创建订单**
   - `POST /api/v1/website/create-order`
   - 参数: userId, packageId, duration

4. **发起支付**
   - `POST /api/v1/website/pay`
   - 参数: orderId, payMethod

5. **支付回调**
   - `POST /api/v1/website/pay-callback`
   - 参数: orderId, status, transactionId

## 部署方案

### Nginx 配置
```nginx
server {
    listen 80;
    server_name www.yunke-crm.com;
    
    root /var/www/website/dist;
    index index.html;
    
    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```
