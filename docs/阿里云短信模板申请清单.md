# 阿里云短信模板申请清单

## 📋 模板申请说明

本文档包含云客CRM系统所需的所有阿里云短信模板。请按照以下清单逐条在阿里云短信服务控制台申请模板。

### 申请注意事项：
1. 阿里云短信变量格式：`${变量名}`
2. 验证码类模板需要选择"验证码"类型
3. 通知类模板需要选择"短信通知"类型
4. 模板内容需要符合阿里云审核规范，不能包含营销推广内容
5. 建议先申请签名，再申请模板
6. 每个模板申请后会获得一个模板CODE，需要记录下来配置到系统中

## 1️⃣ 注册验证码模板

**模板名称：** 注册验证码

**申请类型：** 验证码

**模板内容：**
```
您的注册验证码是：${code}，5分钟内有效，请勿泄露给他人。
```

**变量说明：**
- `code`: 6位数字验证码

**使用场景：** 
- 官网注册时发送手机验证码
- 用于验证用户手机号真实性

**对应文件：** 
- `backend/src/controllers/public/RegisterController.ts`
- `backend/src/services/VerificationCodeService.ts`

## 2️⃣ 账号开通通知（免费试用）

**模板名称：** 账号开通通知

**申请类型：** 短信通知

**模板内容：**
```
尊敬的${tenantName}，您的云客CRM账号已开通！租户编码：${tenantCode}，授权码：${licenseKey}，管理员账号为您的手机号，初始密码：${adminPassword}，套餐：${packageName}，到期日期：${expireDate}。请妥善保管账号信息，登录后请立即修改密码。
```

**变量说明：**
- `tenantName`: 租户名称（企业名称）
- `tenantCode`: 租户编码（短代码）
- `licenseKey`: 授权码（完整授权码）
- `adminPassword`: 初始密码
- `packageName`: 套餐名称
- `expireDate`: 到期日期（格式：2026-12-31）

**重要修改说明：**
- ❌ 已移除 `adminUsername` 变量，改为固定文字"管理员账号为您的手机号"（阿里云不允许手机号作为变量）
- ❌ 已移除 `loginUrl` 变量（阿里云不允许URL作为变量传入）
- ✅ 移除了开头的【】签名符号，签名会自动添加

**使用场景：** 
- 用户选择免费试用套餐完成注册后立即发送
- 包含登录所需的全部信息

**对应文件：** 
- `backend/src/controllers/public/RegisterController.ts` (register方法，仅免费套餐)

## 🔧 配置说明

### 1. 申请流程
1. 登录阿里云控制台
2. 进入"短信服务"
3. 先申请短信签名（如：【云客CRM】）
4. 再逐条申请短信模板
5. 等待审核（通常1-2个工作日）
6. 审核通过后获得模板CODE

### 2. 系统配置

**管理后台配置（推荐）：**
1. 登录Admin后台 → 设置 → 基础配置 → 短信配置
2. 填写阿里云 AccessKey ID、AccessKey Secret、签名
3. 在"短信模板CODE配置"区域，将每个模板CODE填入对应位置
4. 客户通知类模板：直接对应各业务场景
5. 管理员通知类模板：填写 `ADMIN_NOTIFICATION` 后，在「通知服务」中配置接收手机号

**管理员通知渠道配置：**
1. 管理后台 → 设置 → 通知服务 → 通知渠道 → 启用「短信通知」
2. 填写运营管理员手机号（支持多个）
3. 在「通知规则」中为需要的事件启用 sms 渠道

### 3. 模板CODE映射
在 `backend/src/services/AliyunSmsService.ts` 中维护：

```typescript
templates?: {
  VERIFY_CODE?: string              // 注册验证码
  ACCOUNT_ACTIVATION?: string       // 账号开通通知（免费试用）
  PAYMENT_ACTIVATION?: string       // 支付成功账号开通通知
  RENEW_SUCCESS?: string            // 续费成功通知
  PACKAGE_CHANGE?: string           // 套餐变更通知
  QUOTA_CHANGE?: string             // 配额变更通知
  ACCOUNT_SUSPEND?: string          // 账号暂停通知
  ACCOUNT_RESUME?: string           // 账号激活通知
  ACCOUNT_CANCEL?: string           // 账号注销通知
  REFUND_SUCCESS?: string           // 退款成功通知
  EXPIRE_REMIND?: string            // 到期提醒
  EXPIRED_NOTICE?: string           // 过期通知
  ADMIN_NOTIFICATION?: string       // 管理员通用通知
  SUBSCRIPTION_DEDUCT_SUCCESS?: string  // 订阅扣款成功
```

## ⚠️ 注意事项

1. **字数限制**：阿里云短信单条最多500字符，建议控制在300字符以内
2. **变量规范**：变量名只能包含字母、数字、下划线
3. **签名要求**：签名需要与企业资质一致，需要提供营业执照等证明材料
4. **审核时间**：工作日通常1-2天，节假日可能延长
5. **测试环境**：开发环境可以使用固定验证码（123456）进行测试，未配置的模板会模拟发送
6. **发送频率**：同一手机号1分钟内只能发送1条，1小时内最多5条，1天内最多10条
7. **费用说明**：验证码短信约0.045元/条，通知短信约0.045元/条
8. **URL限制**：阿里云不允许URL作为短信变量，登录链接等信息需写为固定文字
9. **多渠道同步**：重要通知（如支付成功、扣款失败、到期提醒）建议同时启用短信+邮件双渠道，确保送达
