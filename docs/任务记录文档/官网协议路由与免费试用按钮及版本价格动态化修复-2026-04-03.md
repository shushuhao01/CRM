# 官网协议路由与免费试用按钮及版本价格动态化修复

**日期**：2026-04-03  
**涉及系统**：官网（website）  
**修改范围**：Register.vue、Navbar.vue、Home.vue

---

## 问题清单

### 问题1：用户协议和隐私政策链接无法正常跳转

- **位置**：`website/src/views/Register.vue` 第201行
- **现象**：注册页面填写信息步骤中，点击《服务协议》和《隐私政策》链接，看不到对应的协议内容
- **原因**：链接使用了 `query 参数`（`/agreement?type=service`），但路由定义为 `路径参数`（`/agreement/:type`），`Agreement.vue` 读取的是 `route.params.type`，导致参数获取为空

### 问题2：导航栏和首页"免费试用"按钮跳转到微信客服

- **位置**：`website/src/components/Navbar.vue` 第38行，`website/src/views/Home.vue` 第31行和第405行
- **现象**：点击右上角"免费试用"按钮和首页英雄区"免费试用7天"按钮，跳转到微信客服页面
- **原因**：按钮的 href 绑定到了 `wechatServiceUrl`（微信客服链接），而不是注册页面

### 问题3：首页底部"选择适合您的版本"价格硬编码

- **位置**：`website/src/views/Home.vue` 第298-343行
- **现象**：SaaS云端版显示固定的 `¥99/月起`，私有部署版显示固定的 `¥9,800起`
- **原因**：价格为硬编码字符串，未从管理后台的套餐API获取动态数据

---

## 修复方案与实施

### 修复1：协议链接路由修正

**文件**：`website/src/views/Register.vue`

**变更**：
- 将 `<router-link to="/agreement?type=service">` 改为 `<router-link to="/agreement/service" target="_blank">`
- 将 `<router-link to="/agreement?type=privacy">` 改为 `<router-link to="/agreement/privacy" target="_blank">`
- 添加 `target="_blank"` 使协议在新标签页打开，不中断注册流程

### 修复2：免费试用按钮跳转到注册页面

**文件**：`website/src/components/Navbar.vue`、`website/src/views/Home.vue`

**变更**：
- **Navbar.vue**：将 `<a :href="wechatServiceUrl">` 改为 `<router-link to="/register?plan=FREE_TRIAL">`，移除不再使用的 `wechatServiceUrl` 变量
- **Home.vue 英雄区**：将"免费试用7天"按钮从 `<a :href="wechatServiceUrl">` 改为 `<router-link to="/register?plan=FREE_TRIAL">`
- **Home.vue CTA区**：将底部"免费试用7天"按钮改为 `<router-link to="/register?plan=FREE_TRIAL">`；"预约演示"保留为微信客服链接
- 用户点击后直接进入注册页面，自动预选"7天免费试用"套餐

### 修复3：首页版本价格动态化

**文件**：`website/src/views/Home.vue`

**变更**：
- 引入 `getPackages` API 和 `Package` 类型
- 新增 `packagesData` 响应式变量，在 `onMounted` 中加载套餐数据
- 新增 `saasMinPrice` 计算属性：从SaaS付费套餐中取最低价，兜底值99
- 新增 `privateMinPrice` 计算属性：从私有部署套餐中取最低价，兜底值9,800
- 模板中的硬编码价格改为使用计算属性动态渲染

## 技术细节

### 路由对应关系

| 路由定义 | 正确链接 | 参数读取方式 |
|---------|---------|------------|
| `/agreement/:type` | `/agreement/service` | `route.params.type` |
| `/agreement/:type` | `/agreement/privacy` | `route.params.type` |

### 注册页面预选逻辑

Register.vue 的 `onMounted` 中已有处理 `route.query.plan` 的逻辑：
- 当 `plan=FREE_TRIAL` 时，自动设置 `selectedPlan = 'FREE_TRIAL'` 和 `planType = 'saas'`
- 无需额外修改注册页面逻辑

### 价格兜底策略

当API请求失败或返回空数据时：
- SaaS最低价显示 99（基础版默认价格）
- 私有部署最低价显示 9,800（标准版默认价格）
