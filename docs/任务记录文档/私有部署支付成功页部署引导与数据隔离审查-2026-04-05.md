### 新增功能详细说明

#### 1. 智能引导问卷（5个问题分支）

```
Q1: 您是否有阿里云、腾讯云等云服务器？
├─ ✅ 有云服务器
│  ├─ Q2: 是否有已备案的域名？ → [有/只有IP]
│  └─ Q3: 是否安装/打算安装宝塔面板？ → [宝塔/纯命令行]
│     ├─ 宝塔 → 推荐「宝塔面板」标签页
│     └─ 命令行 → 推荐「纯Linux」标签页
└─ ❌ 没有云服务器
   ├─ Q2: 有独立电脑做本地部署吗？
   │  ├─ 有 → Q3: 会不会下载安装 Node.js 和 MySQL？
   │  │  ├─ 会 → 推荐「本地电脑」标签页
   │  │  └─ 不确定 → 推荐「本地电脑」+ 显示客服协助入口
   │  └─ 没有 → 建议购买服务器 + 显示客服协助入口
```

#### 2. 三个部署标签页

| 标签 | 适用场景 | 一键部署 | 常规部署 |
|------|---------|---------|---------|
| 🖥️ 本地电脑 | 快速体验/测试 | `build-local.bat`/`build-local.sh` | 手动npm install + build |
| 🚀 宝塔面板（推荐） | 正式生产部署 | `deploy.sh` 一键脚本 | 手动安装依赖+构建+PM2 |
| 🐧 纯Linux | 有运维经验用户 | `deploy.sh` 一键脚本 | 手动全流程 |

每个标签页包含：
- 部署方式切换（⚡一键部署 / 📝常规部署）
- 逐步骤卡片式教程
- 代码块带复制按钮
- OS系统切换（Ubuntu/CentOS）
- 关键步骤高亮标注

#### 3. 客服链接集成

- **数据来源**：从管理后台官网配置API `GET /api/v1/public/website-config` 获取 `customerServiceUrl`
- **降级策略**：API获取失败时使用默认微信客服链接 `https://work.weixin.qq.com/kfid/kfc461ca9f5b45c8d25`
- **展示位置**：
  - 引导问卷中"需要协助"时显示绿色客服按钮
  - 底部操作按钮区的"联系客服协助"按钮
  - 页面底部帮助提示中的"联系在线客服"链接

#### 4. 其他增强
- 一键复制授权信息按钮
- 常用运维命令速查表（pm2 list / logs / restart / 更新部署）
