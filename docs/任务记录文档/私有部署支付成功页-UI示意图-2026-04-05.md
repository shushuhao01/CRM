## 引导问卷分支流程

```
                     Q1: 有没有云服务器？
                    /                    \
               ✅ 有                    ❌ 没有
              /    \                   /        \
         Q2:域名?  Q3:宝塔?      Q2:有电脑?
         /    \     /      \       /       \
     有域名  纯IP  宝塔  纯命令  ✅有     ❌没有
                   │      │      │         │
                   ▼      ▼      ▼         ▼
              推荐宝塔  推荐Linux  Q3:会装?  推荐买服务器
              ▸标签页   ▸标签页   /    \    + 联系客服
                              会    不会
                              │      │
                              ▼      ▼
                         推荐本地  推荐本地
                         ▸标签页   + 联系客服
```

### 5种推荐结果

| 路径 | 推荐部署方式 | 显示客服入口 |
|------|------------|------------|
| 有服务器 → 宝塔 | 🚀 宝塔面板标签页 | ❌ |
| 有服务器 → 纯命令行 | 🐧 纯Linux标签页 | ❌ |
| 无服务器 → 有电脑 → 会装 | 🖥️ 本地电脑标签页 | ❌ |
| 无服务器 → 有电脑 → 不确定 | 🖥️ 本地电脑标签页 | ✅ |
| 无服务器 → 无设备 | 建议购买服务器 | ✅ |
