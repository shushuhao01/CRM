## 三、套餐订阅配置说明

### 套餐管理中的订阅配置位置
- 路径：管理后台 → 租户客户 → 套餐管理 → 编辑/新增套餐
- **显示条件**：套餐类型为 SaaS云端版 且 月付价格 > 0 且 非试用套餐
- 免费试用套餐和私有部署套餐不支持订阅模式（这是正确的业务逻辑）

### 订阅配置项
| 配置项 | 说明 |
|--------|------|
| 支持订阅（开关） | 开启后官网支付时默认推荐订阅模式 |
| 订阅渠道 | 全部支持 / 仅微信代扣 / 仅支付宝扣款 |
| 订阅优惠折扣 | 百分比（如10=9折），0表示无优惠 |
| 价格预览 | 实时显示正常月付价 vs 订阅价格 vs 年累计节省 |

### 如何开启套餐订阅
1. 进入管理后台 → 租户客户 → 套餐管理
2. 选择一个 **SaaS付费套餐**（非免费试用），点击"编辑"
3. 滚动到"🔄 订阅付费配置"区块
4. 开启"支持订阅"开关
5. 选择支持的订阅渠道和优惠折扣
6. 点击"确定"保存

## 四、支付配置说明

### 商户号配置
- 路径：管理后台 → 支付管理 → 支付配置
- 配置好微信/支付宝商户号后，系统会自动调用对应的支付接口
- **委托代扣/周期扣款权限**：需要在微信商户平台/支付宝开放平台申请开通
  - 微信：需开通"委托代扣"产品权限
  - 支付宝：需签约"周期扣款"产品

### 系统自动处理
- 配置好商户号且套餐开启订阅后，系统自动在注册/付费页面展示订阅选项
- 签约成功后，系统自动执行定时扣款（每天01:00）
- 扣款失败自动重试（每天14:00，最多3次，7天宽限期）
