## 一、任务概述

为云客CRM官网新增 **会员中心** 和 **订阅付费**（微信委托代扣/支付宝周期扣款）功能，包含：
- 管理后台套餐配置增加订阅开关和渠道设置
- 官网注册流程增加订阅支付和免费试用自动续费
- 独立的会员中心系统（登录、概览、企业信息、订阅管理、账单、授权、密码、续费升级）
- 微信/支付宝委托代扣全链路（签约、扣款、解约、定时任务、宽限期重试）

## 七、认证体系

```
三套完全独立的JWT认证：

1. CRM用户认证
   中间件: authenticateToken → req.user
   Secret: JWT_SECRET (crm-secret-key)
   Token有效期: 7天(生产) / 365天(开发)

2. Admin管理员认证
   中间件: adminAuthMiddleware → req.adminUser
   Secret: JWT_SECRET (admin-secret-key)

3. 会员中心认证（新增）
   中间件: memberAuth → req.memberTenant
   Secret: MEMBER_JWT_SECRET (member-center-secret-key-2026)
   Token有效期: 7天
```

## 八、定时任务

| 任务名 | 执行时间 | 说明 |
|--------|----------|------|
| `subscription-auto-deduct` | 每天01:00 | 扫描到期订阅执行自动扣款 |
| `subscription-deduct-retry` | 每天14:00 | 扣款失败重试（3次+7天宽限期） |

### 扣款失败宽限期策略
```
Day 0: 首次扣款失败 → 1天后重试
Day 1: 第1次重试失败 → 2天后重试
Day 3: 第2次重试失败 → 2天后重试
Day 5: 第3次重试失败 → 订阅过期
Day 7: 宽限期结束
```

## 九、管理后台操作指南

### 启用套餐订阅
1. 进入管理后台 → 租户管理 → 套餐管理
2. 编辑任意SaaS付费套餐
3. 在"订阅付费配置"区块：
   - 开启"支持订阅"开关
   - 选择订阅渠道（全部/仅微信/仅支付宝）
   - 设置订阅优惠折扣（如10=9折）
4. 保存后，官网注册支付时会默认推荐订阅模式

### 验证效果
- 管理后台套餐卡片显示绿色"🔄 支持订阅"标签
- 官网注册步骤3显示"订阅付费/正常支付"切换
- 免费试用注册显示"到期自动续费"勾选框
