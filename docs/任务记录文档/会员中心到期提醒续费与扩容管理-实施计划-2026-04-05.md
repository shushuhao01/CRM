## 需求概述

### 功能1：套餐到期提醒与快捷续费
- 会员中心首页（Dashboard），当有效期剩余 < 20% 时弹出提醒
- 显示当前套餐剩余天数，提供"立即续费"按钮
- 点击弹出套餐详情和价格，生成微信/支付宝收款二维码
- 支付成功后自动延长有效期

### 功能2：扩容管理（用户数 + 存储空间独立升级）
- 管理后台「套餐管理」一级菜单下新增「扩容管理」子菜单，配置扩容价格
- 会员中心新增「扩容管理」菜单页面，用户可自主购买额外用户数/存储空间
- 扩容费用叠加到续费费用中
- 扩容成功后 CRM、管理后台、会员中心同步生效

## 阶段1：数据库 + 后端基础（扩容价格配置 + 到期提醒增强）

### 数据库变更
1. **新建 `capacity_price_configs` 表**（扩容价格配置）
   - `id` VARCHAR(36) PK
   - `type` ENUM('user','storage') — 扩容类型
   - `billing_cycle` ENUM('monthly','yearly','follow_package') — 计费周期
   - `unit_price` DECIMAL(10,2) — 单价（每用户/每GB）
   - `min_qty` INT DEFAULT 1 — 最小购买量
   - `max_qty` INT DEFAULT 100 — 最大购买量
   - `description` VARCHAR(255) — 描述
   - `is_active` TINYINT DEFAULT 1
   - `created_at`, `updated_at`

2. **新建 `capacity_orders` 表**（扩容订单）
   - `id` VARCHAR(36) PK
   - `order_no` VARCHAR(64) UNIQUE
   - `tenant_id` VARCHAR(36)
   - `type` ENUM('user','storage')
   - `quantity` INT
   - `unit_price` DECIMAL(10,2)
   - `total_amount` DECIMAL(10,2)
   - `billing_cycle` VARCHAR(20)
   - `pay_type` VARCHAR(20)
   - `status` ENUM('pending','paid','closed') DEFAULT 'pending'
   - `trade_no` VARCHAR(128)
   - `paid_at` DATETIME
   - `created_at`

3. **tenants 表新增字段**
   - `extra_users` INT DEFAULT 0
   - `extra_storage_gb` INT DEFAULT 0
