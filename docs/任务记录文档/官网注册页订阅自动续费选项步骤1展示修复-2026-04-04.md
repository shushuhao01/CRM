# 官网注册页订阅自动续费选项步骤1展示修复

> 日期：2026-04-04
> 模块：官网（website）→ 注册页（Register.vue）
> 关联：管理后台套餐管理订阅配置

---

## 📋 问题描述

管理后台的套餐管理中，对应套餐已配置好「支持订阅」的设置（`subscription_enabled`、`subscription_discount_rate`、`subscription_channels` 等字段），但在官网注册页面中：

1. **订阅选项仅在步骤3（支付确认页）才显示**，作为"支付模式切换"出现
2. 用户在步骤1选择套餐时**看不到订阅相关信息**，无法提前感知订阅优惠
3. 缺少默认勾选逻辑，不符合业务预期的"默认引导订阅"流程

### 期望行为

- 在步骤1「选择套餐」的月付/年付切换区域**下方**，展示「到期自动续费订阅」区域
- **默认勾选**，展示订阅优惠价和优势说明
- 勾选状态下点击「下一步」→ 步骤3自动进入**签约支付**模式
- 取消勾选后点击「下一步」→ 步骤3进入**普通扫码支付**模式
- 如果套餐未配置订阅支持（`subscription_enabled = false`），则**不显示**此区域

## 🔧 修复方案（含二次优化）

### 修改文件

- `website/src/views/Register.vue`

### 第一次修复：基础功能

#### 1. 模板（Template）- 步骤1新增订阅选项区域

在月付/年付切换（`.billing-toggle`）和私有部署方案之间，插入订阅自动续费选项模块。

**显示条件**：
- `planType === 'saas'`：仅SaaS云端版
- `selectedPlan !== 'FREE_TRIAL'`：非免费试用
- `currentPkgSubscriptionEnabled`：当前套餐后台配置了订阅支持

#### 2. 脚本（Script）- 响应式状态与同步

```typescript
const subscriptionChecked = ref(true) // 默认勾选

// 订阅勾选状态 → 同步支付模式
watch(subscriptionChecked, (val) => {
  payMode.value = val ? 'subscription' : 'normal'
})

// 套餐订阅支持状态变化时（切换套餐），自动重置订阅勾选
watch(currentPkgSubscriptionEnabled, (enabled) => {
  subscriptionChecked.value = enabled
  payMode.value = enabled ? 'subscription' : 'normal'
})
```

#### 4. 年付订阅支持

**新增计算属性**：

```typescript
// 年付订阅价格（年付优惠 + 订阅折扣叠加）
const subscriptionYearlyPrice = computed(() => {
  const yearly = getYearlyPrice(selectedPlan.value)
  if (currentPkgSubscriptionDiscountRate.value > 0) {
    return Math.round(yearly * (1 - currentPkgSubscriptionDiscountRate.value / 100))
  }
  return yearly
})

// 订阅额外节省金额
const subscriptionSaving = computed(() => {
  if (billingCycle.value === 'yearly') {
    return getYearlyPrice(selectedPlan.value) - subscriptionYearlyPrice.value
  }
  return getMonthlyPrice(selectedPlan.value) - Number(subscriptionPrice.value)
})
```

**价格逻辑示例**（基础版 ¥199/月，年付送2个月 ¥1990/年，订阅折扣10%）：

| 模式 | 原价 | 订阅价 | 额外省 |
|------|------|--------|--------|
| 月付 | ¥199/月 | ¥179/月 | ¥20/月 |
| 年付 | ¥1990/年 | ¥1791/年 | ¥199/年 |

**模板中的双模式显示**：
- 月付时：`订阅价 ¥179/月 省10% 原价 ¥199/月`
- 年付时：`订阅价 ¥1,791/年 再省10% 年付价 ¥1,990/年`
- 底部：`订阅自动续费再省 ¥20/月` 或 `订阅自动续费再省 ¥199/年`

## ✅ 逻辑流程

```
步骤1：选择套餐
  ├── 选择SaaS付费套餐（basic/pro/enterprise）
  ├── 选择月付/年付
  ├── 【订阅区域】到期自动续费（默认勾选）
  │   ├── 月付 → 显示月订阅优惠价 & 月省金额
  │   ├── 年付 → 显示年订阅优惠价（叠加年付折扣） & 年省金额
  │   ├── 勾选 → payMode = 'subscription'
  │   └── 取消勾选 → payMode = 'normal'（整体置灰）
  └── 点击"下一步"

步骤3：支付/签约
  ├── payMode = 'subscription' → 签约支付（按月/年自动扣款）
  └── payMode = 'normal' → 普通支付（一次性付款）
```
