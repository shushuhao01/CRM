# Admin套餐显示修复与官网信息动态化配置 - 2026-04-03

## 任务概述

修复两个问题：
1. **套餐管理**：7天免费试用套餐价格显示为"¥0.00/永久"，应显示为"免费/7天试用"
2. **官网信息动态化**：在Admin管理后台基础配置中新增"官网信息"选项卡，将官网底部的版权/备案/联系方式/客服信息从硬编码改为从管理后台动态获取

## 问题分析

### 问题1：试用套餐显示"永久"

**现象**：Admin后台套餐管理页面，7天免费试用套餐价格单位显示为 `/永久`（截图红框处）。

**根因**：`Packages.vue` 的 `getBillingText()` 函数将 `billing_cycle='once'` 固定映射为 `'永久'`，但试用套餐虽然 `billing_cycle='once'`，实际有 `duration_days=7` 的时间限制。官网 `Pricing.vue` 已正确处理（显示 `/ 7天`），但Admin后台未区分。

### 问题2：官网信息硬编码

**现象**：官网 `Footer.vue` 中版权信息 `© 2025 云客CRM`、电话号码 `13570727364`、客服链接、二维码路径均为硬编码；`CustomerService.vue` 中客服链接和二维码也是硬编码。修改这些信息需要改源码重新部署。

**修复方案**：
- Admin基础配置新增"官网信息"选项卡，集中管理官网展示所需的全部动态信息
- 后端新增公开API供官网获取这些配置
- 官网前端组件从API获取配置后动态渲染

## 详细修改说明

### 1. 试用套餐显示修复

**修改前**：
```html
<span class="unit">/{{ getBillingText(pkg.billing_cycle) }}</span>
```
所有套餐统一使用 `billing_cycle` 映射（`once → 永久`），试用套餐误显"永久"。

**修改后**：
```html
<span class="unit" v-if="pkg.is_trial && pkg.price === 0">/{{ pkg.duration_days || 7 }}天试用</span>
<span class="unit" v-else>/{{ getBillingText(pkg.billing_cycle) }}</span>
```
试用套餐显示实际有效天数。

### 2. Admin Config.vue "官网信息"选项卡

新增的表单字段（`websiteConfig` 对象）：

| 字段 | 类型 | 用途 |
|------|------|------|
| `customerServiceUrl` | string | 在线客服链接（企业微信客服等） |
| `serviceQRCode` | string | 客服二维码图片URL（支持上传） |
| `servicePhone` | string | 客服电话 |
| `serviceEmail` | string | 客服邮箱 |
| `workingHours` | string | 工作时间文字 |
| `brandSlogan` | string | 品牌标语 |

这些字段保存在 `admin_system_config` 的 `websiteConfig` 子对象中，与现有的 `copyrightText`、`icpNumber`、`policeNumber` 等顶层字段协同工作。

### 3. 官网前端动态化策略

**降级机制**：所有动态字段都有默认降级值，API获取失败时不影响官网正常显示：

| 字段 | 降级默认值 |
|------|-----------|
| `customerServiceUrl` | `https://work.weixin.qq.com/kfid/kfc461ca9f5b45c8d25` |
| `servicePhone` | `13570727364` |
| `serviceQRCode` | `/images/kefuQR.png` |
| `workingHours` | `周一至周五 9:00-18:00` |
| `copyrightText` | `© 当前年份 云客CRM. All rights reserved.` |
| `companyName` | `云客CRM` |
| `brandSlogan` | `智能销售管理系统，助力企业高效增长` |

**请求缓存**：`website-config.ts` 内置内存缓存，页面生命周期内只请求一次API，Footer和CustomerService共享同一份缓存数据。

## 测试验证

1. **试用套餐**：登录Admin后台 → 租户客户 → 套餐管理 → 确认7天免费试用套餐显示"免费/7天试用"而非"¥0.00/永久"
2. **官网信息配置**：Admin后台 → 模块服务 → 基础配置 → "官网信息"选项卡 → 填写版权/备案/客服信息 → 保存
3. **官网同步验证**：访问官网 → 底部版权/备案信息已更新 → 客服面板二维码和链接已更新 → 电话号码已更新
4. **降级验证**：清空Admin官网配置 → 官网仍能正常显示默认值

## 注意事项

- 公开API `GET /api/v1/public/website-config` 不需要认证，适合官网直接调用
- `serviceQRCode` 优先使用 `websiteConfig.serviceQRCode`，如果未设置则降级到 `contactQRCode`（基本设置中的二维码）
- 官网前端使用内存缓存，避免同一页面多个组件重复请求API
- 备案链接使用了正式的备案查询网站URL：ICP → `beian.miit.gov.cn`，公安 → `beian.gov.cn`
