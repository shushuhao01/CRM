## 二、阶段2 — CRM端改造 + 后端API + 数据库 ✅

### 2.1 数据库与实体
- **SmsTemplate** 新增8个字段：vendorTemplateCode、vendorStatus、vendorSubmitAt、vendorRejectReason、adminReviewer、adminReviewAt、adminReviewNote、isPreset
- **SmsRecord** 新增 senderPhone 字段
- **status 枚举扩展**：pending_admin → pending_vendor → active，新增 withdrawn/deleted 状态

### 2.2 后端 sms.ts 路由改造
| API | 方法 | 功能 |
|-----|------|------|
| `/templates/available` | GET | 获取可用模板（active + 预设模板） |
| `/templates/my-applications` | GET | 获取我的模板申请列表 |
| `/templates/apply` | POST | 申请短信模板（status=pending_admin） |
| `/templates/:id/withdraw` | POST | 撤销模板申请（仅pending_admin可撤） |
| `/templates/:id` | DELETE | 删除已拒绝/已撤销的模板 |
| `/variable-docs` | GET | 获取模板变量文档（100+变量，12大分类） |

### 2.3 CRM前端
- **SmsManagement.vue**：注释模板管理/审核按钮 → 新增"申请模板"和"查看模板"入口
- **SmsTemplateDetail.vue**（929行全新页面）：
  - Tab1：可用模板列表（卡片网格展示，搜索/分类筛选）
  - Tab2：我的申请（状态筛选、撤销、删除、拒绝原因展示）
  - Tab3：变量文档（分组展示、搜索、一键复制变量名）
- **通知集成**：模板申请时发送 MessageType.SMS_TEMPLATE_APPLIED 通知管理员
- **API层**：sms.ts 新增8个API函数完整覆盖

### 2.4 配置下发改造
- **Distribute.vue**：SMS卡片"编辑配置"改为"前往配置"，跳转至 `/sms-management/crm-config`

## 三、阶段3 — 管理后台短信管理模块 ✅

### 3.1 菜单与路由（6个二级页面）
```
📱 短信管理 (一级菜单)
  ├── 📋 模板审核     /sms-management/template-review
  ├── 📑 模板管理     /sms-management/templates
  ├── 📊 发送记录     /sms-management/records
  ├── ⚙️ CRM配置     /sms-management/crm-config
  ├── 📈 统计总览     /sms-management/statistics
  └── 💰 额度套餐     /sms-management/quota-packages
```

### 3.2 各页面功能

| 页面 | 行数 | 核心功能 |
|------|------|---------|
| TemplateReview.vue | 357行 | 四阶段审核（approve/reject/activate）、统计卡片、审核弹窗含模板CODE填入 |
| Templates.vue | 309行 | 预设模板创建、全租户模板查看、筛选（类型/分类/关键词）、编辑/删除 |
| Records.vue | 188行 | 跨租户发送记录、日期/状态筛选、详情弹窗 |
| CrmConfig.vue | 188行 | 短信服务商配置（阿里云/腾讯云/华为云）、密钥/签名/限额/管控开关 |
| Statistics.vue | 350行 | 核心指标卡片、模板状态分布、近7天发送趋势图 |
| QuotaPackages.vue | 655行 | 套餐CRUD、单价管理、订单列表、退款预览弹窗、日期筛选 |

### 3.3 后端 admin/sms-management.ts（395行）
| API | 方法 | 功能 |
|-----|------|------|
| `/template-review` | GET | 跨租户模板审核列表（含关键词搜索） |
| `/template-review/:id` | POST | 管理员审核操作（approve/reject/activate） |
| `/templates` | GET | 全部模板列表（预设+租户，含筛选分页） |
| `/templates` | POST | 创建预设模板（isPreset=1） |
| `/templates/:id` | PUT | 更新模板（含CODE填入自动激活） |
| `/templates/:id` | DELETE | 逻辑删除模板 |
| `/records` | GET | 跨租户发送记录（含日期/状态筛选） |
| `/statistics` | GET | 全平台统计（模板分布、发送量、趋势） |

### 3.4 API封装 smsManagement.ts（133行）
完整覆盖所有管理后台短信管理接口，含类型定义。
