## 六、操作手册摘要

### 6.1 首次接入企业微信

1. 进入CRM系统 → 企微管理 → 企微配置
2. 右上角点击「扫码授权」或「添加配置」
3. **扫码授权(SaaS推荐)**: 点击生成二维码 → 企微管理员扫码 → 系统自动完成
4. **手动添加**: 填写企业ID(CorpID)、应用Secret、AgentID → 测试连接 → 保存

### 6.2 会话存档开通

1. 企微管理 → 会话存档 → 套餐购买Tab
2. 选择席位数量和时长
3. 完成支付后在「存档设置」Tab配置拉取间隔、可见性等
4. 在「席位管理」中选择生效成员

### 6.3 获客助手使用

1. 企微管理 → 获客助手
2. 创建获客链接，选择承接成员
3. 可设置成员权重分配
4. 查看渠道分析统计
