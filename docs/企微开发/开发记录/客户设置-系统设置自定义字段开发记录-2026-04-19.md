# 客户设置 - 系统设置自定义字段开发记录

> 开发日期：2026-04-19
> 模块：系统管理 > 系统设置 > 客户设置

## 一、需求概述

在系统管理的系统设置中新增"客户设置"标签选项卡，功能包括：

1. **内置字段配置**：支持设置新增客户表单字段的启用/隐藏、必选/非必选
2. **自定义字段管理**：预设10个自定义字段槽位，支持文本、数字、日期、下拉、多选等类型
3. **客户详情同步**：客户详情页基本信息卡片底部同步显示已配置的自定义字段
4. **列表导出同步**：客户列表批量导出和列设置中同步显示自定义字段
5. **快速跳转**：新增客户页面提供管理员快速跳转到客户设置的图标入口

## 二、技术方案

### 2.1 数据库设计

- **配置存储**：使用已有的 `system_configs` 表，新增 `configKey='customerFieldConfig'`, `configGroup='customer_settings'` 记录
- **客户数据存储**：在 `customers` 表新增 `custom_fields` JSON 列，存储自定义字段值

### 2.2 后端 API

- `GET /api/v1/system/customer-field-config` - 获取客户字段配置
- `PUT /api/v1/system/customer-field-config` - 更新客户字段配置（管理员）

### 2.3 前端组件

- `src/stores/customerFieldConfig.ts` - 客户字段配置 Store
- `src/views/Settings/CustomerSettings.vue` - 客户设置页面组件
- 修改 `src/views/System/Settings/index.vue` - 注册客户设置标签页
- 修改 `src/views/Customer/Add.vue` - 新增客户表单集成自定义字段
- 修改 `src/views/Customer/Detail.vue` - 客户详情集成自定义字段
- 修改 `src/views/Customer/List.vue` - 列表和导出集成自定义字段
- 修改 `src/utils/export.ts` - 导出工具支持自定义字段
