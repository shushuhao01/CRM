| 功能 | 说明 | 状态 |
|------|------|------|
| 统计概要 | 已授权数/活跃/待处理/已取消 | ✅ UI完成 |
| 授权列表 | 企业名称、CorpID、关联租户、授权时间、状态 | ✅ UI完成 |
| 生成授权链接 | 支持通用链接/指定租户链接，权限范围(完整/基础)，有效期(1/7/30天) | ✅ UI完成 |
| 关联租户 | 将已授权企业关联到CRM租户 | ✅ UI完成 |
| 取消授权 | 取消某企业的应用授权 | ✅ UI完成 |

**完整流程**：
1. 管理员在「应用配置」Tab配置好Suite ID/Secret等
2. 在「授权管理」Tab点击"生成授权链接"
3. 将授权链接发给目标企业管理员
4. 企业管理员扫码授权 → 企业出现在授权列表(pending状态)
5. 管理员点击"关联租户"将企业与CRM租户绑定
6. 状态变为active → 该租户CRM系统即可使用企微功能

#### Tab3: 回调配置 ✅ 前端完备（已优化分页）

| 功能 | 说明 | 状态 |
|------|------|------|
| 回调URL | 自动生成 `{origin}/api/v1/wecom/suite/callback` | ✅ |
| Token | 用于验证回调消息来源，支持随机生成 | ✅ |
| EncodingAESKey | 用于加解密回调消息，支持随机生成 | ✅ |
| 回调日志 | 展示回调事件接收记录 | ✅ 已添加分页(默认10条/页) |

**回调配置的作用和使用方法**：

回调是企微服务器主动推送事件到我们服务器的机制。配置步骤：
1. 复制回调URL、Token、EncodingAESKey
2. 登录企微服务商后台 → 应用管理 → 找到我们的第三方应用
3. 在"数据回调URL"/"指令回调URL"中填入这些值
4. 企微服务器会定时推送 `suite_ticket`（每10分钟）、授权变更事件等
5. 我们的后端接收这些回调来自动更新 Suite Ticket、处理授权/取消授权事件

**回调日志**可查看每次回调的：时间、事件类型（如`suite_ticket`/`create_auth`/`cancel_auth`）、处理状态（成功/失败）、详情

## 五、侧边栏话术预览SQL错误修复

### 问题
侧边栏内置应用预览中，新建/编辑话术选择颜色后保存报SQL语法错误：
```
You have an error in your SQL syntax... near ', `color` = '#ab47bc', `updated_at` = CURRENT_TIMESTAMP WHERE `id` IN (13)'
```

### 原因
`PUT /wecom/scripts/:id` 更新路由未调用 `ensureScriptTables()` 进行列迁移检查，导致数据库表缺少 `color`/`attachments`/`tags` 等后期新增列时，TypeORM生成的UPDATE SQL失败。

### 修复
**文件**: `backend/src/routes/wecom/scripts.ts`

1. `PUT /scripts/:id` 路由添加 `await ensureScriptTables()` 调用
2. `ensureScriptTables()` 迁移逻辑增强：
   - 使用反引号 `` ` `` 包裹列名避免MySQL保留字冲突
   - 新增 `use_count`/`ai_rewrite_enabled`/`is_enabled`/`attachments`/`tags` 列的自动迁移
   - 迁移失败时重置 `tableMigrated` 标志以便下次重试
   - 添加迁移日志便于排查

## 九、服务商应用管理完整说明

### Q: 配置好应用后，租户就能在CRM端扫码授权了吗？

**完整流程**：
1. ✅ 在「应用配置」Tab填写 Suite ID、Suite Secret、服务商CorpID/Secret → 保存
2. ✅ 在「回调配置」Tab获取回调URL/Token/AESKey → 填到企微服务商后台
3. ✅ 在「授权管理」Tab生成授权链接/二维码 → 发给目标企业管理员扫码
4. ✅ 企业管理员扫码授权 → 企业出现在授权列表
5. ✅ 管理员"关联租户" → 该租户CRM系统激活企微功能
6. ✅ 从此我们可调用该企业的企微API：获取通讯录、客户、群、会话存档等

### Q: 授权链接怎么用？

**两种用法**：
- **推广模式**：生成通用链接/二维码，放在官网/宣传材料中，任何企业都能扫码安装
- **定向模式**：指定租户链接，发给特定客户让他扫码，自动关联到对应租户
