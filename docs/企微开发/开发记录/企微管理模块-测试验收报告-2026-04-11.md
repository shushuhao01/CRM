## 🐛 验收中发现并修复的问题

### Bug 1: `CustomerDetail.vue` 模板中 `window` 不可访问 (已修复)

**位置:** `src/views/WecomSidebar/CustomerDetail.vue` 第201行
**问题:** Vue 3 模板中不能直接访问 `window` 全局对象
```html
<!-- 修复前 -->
<el-button @click="window.location.reload()">刷新</el-button>
<!-- 修复后 -->
<el-button @click="reloadPage">刷新</el-button>
```
**修复:** 新增 `reloadPage()` 方法封装 `window.location.reload()`

## ⚠️ 已知的非阻塞TypeScript类型注解问题

`ChatArchive.vue` 中有若干 TS2339 类型错误（显示 `Property 'list' does not exist on type 'AxiosResponse'`），这是因为项目的 `request.ts` 拦截器在运行时会自动解包 `response.data.data`（第212行 `return data`），但 TypeScript 静态分析仍认为返回类型是 `AxiosResponse`。

**影响:** 仅影响 IDE 类型检查，**不影响运行时功能**。
**建议:** 后续可通过给API函数添加泛型返回类型来消除。
