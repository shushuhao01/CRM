## 二、完成内容

### 2.1 示例数据新增（useWecomDemo.ts）

| 数据常量 | 条数 | 说明 |
|---------|------|------|
| `DEMO_SENSITIVE_WORDS` | 12条 | 覆盖5个分组（售后相关/负面情绪/竞品信息/合规风控/监管风险），含命中次数、级别 |
| `DEMO_SENSITIVE_HITS` | 8条 | 命中记录覆盖4种状态（pending/reviewed/flagged/ignored） |
| `DEMO_SENSITIVE_STATS` | 1组 | 总词数/已启用/累计命中/待审核/分组统计 |
| `DEMO_QUALITY_RULES` | 5条 | 覆盖4种类型（响应时长/消息数量/关键词/情绪分析），含触发次数 |
| `DEMO_QUALITY_INSPECTIONS` | 8条 | 覆盖4种状态（excellent/normal/violation/pending） |
| `DEMO_QUALITY_STATS` | 1组 | 总质检数/优秀/正常/违规/平均分 |
| `DEMO_ARCHIVE_SETTINGS` | 1组 | 完整存档配置（可见性/拉取设置/存储/质检/RSA） |

### 2.2 敏感词管理 Tab 示例UI

- **统计卡片**: 敏感词总数、已启用数、累计命中数、待审核数
- **内部3个子Tab**:
  - **敏感词列表**: 表格展示所有敏感词，支持分组筛选，操作列含编辑/删除按钮，状态开关
  - **命中记录**: 展示命中上下文、发送人、状态(待审核/已审核/已标记/已忽略)，支持状态筛选
  - **分组统计**: 按分组展示敏感词数、累计命中数、命中占比进度条

### 2.3 质检规则 Tab 示例UI

- **统计卡片**: 总质检数、优秀、正常、违规、平均分
- **内部2个子Tab**:
  - **规则管理**: 表格展示质检规则，含类型标签、分值(正值绿色/负值红色)、触发次数、状态开关
  - **质检记录**: 展示员工/客户/评分/状态/违规类型标签，支持状态筛选

### 2.4 存档设置 Tab 示例UI

- **授权状态卡片**: 绿色边框，显示"已开通(示例)"，到期时间、席位用量，Secret/RSA私钥状态标签
- **席位用量条**: 进度条展示已使用/总席位
- **完整设置表单**:
  - 成员可见性（仅自己/本部门/全部可见）
  - 拉取设置（间隔/模式/保留天数）
  - 存储设置（本地/OSS）
  - 质检设置（自动质检开关）
  - RSA公钥配置

### 2.5 交互行为
- 所有操作按钮点击后弹出 `ElMessage.info('示例模式：XXX — 授权企微后可执行真实操作')`
- 筛选下拉框可实时过滤示例数据
- 开关可响应点击（仅改变UI状态，不调用API）
