## 后端API对接说明

所有功能已正确对接后端API路由：
- `GET /wecom/service-accounts` - 客服账号列表
- `POST /wecom/service-accounts` - 创建客服（调用企微 kf/account/add）
- `PUT /wecom/service-accounts/:id` - 更新客服配置
- `POST /wecom/service-accounts/sync` - 从企微同步账号
- `GET /wecom/kf-sessions` - 会话记录（支持分页、筛选）
- `POST /wecom/kf-sessions/sync` - 同步会话消息
- `GET /wecom/kf-stats` - 数据统计
- `GET /wecom/quick-replies` - 快捷回复列表
- `POST/PUT/DELETE /wecom/quick-replies` - CRUD

配置企业微信授权后，所有数据将从真实API加载，示例数据自动消失。

## 企业微信客服API参考

- 创建客服账号: `POST /cgi-bin/kf/account/add`
- 获取账号列表: `POST /cgi-bin/kf/account/list`
- 同步消息: `POST /cgi-bin/kf/sync_msg`
- 发送消息: `POST /cgi-bin/kf/send_msg`
- 获取客服链接: `POST /cgi-bin/kf/add_contact_way`

详见: https://developer.work.weixin.qq.com/document/path/94638

## 第二轮优化 - UI风格统一（2026-04-17）

### 问题
数据统计用了丑陋的柱状图，实时工作台卡片风格不统一，与客户群群数据页面风格不一致。

### 修复内容

1. **统计卡片全部统一为群数据风格** (`stats-grid` + `stats-card` + 颜色主题)
   - 实时工作台：6列卡片，图标+数值+标签横排
   - 数据统计：4列8卡片，群数据同款14px圆角+hover动效

2. **数据统计趋势图改用 TrendLineChart 曲线图**
   - 删除所有柱状图（`kf-trend-bar` 相关代码）
   - 引入 `TrendLineChart` 组件（ECharts平滑曲线+渐变填充）
   - 支持切换 会话/消息 两种曲线

3. **排行表改用排名徽章** (`rank-badge` + `.gold` 前三名金色)

4. **满意度/响应时长分布改用 chart-section 白色圆角卡片包裹**

5. **删除所有旧样式**：`kf-stats-overview`、`kf-overview-item`、`workspace-stats`、`ws-stat-card`、旧 `stats-cards` 嵌套SCSS

### 样式完全统一
现在微信客服页面与客户群群数据页面使用完全相同的：
- `.stats-grid` / `.stats-card` 卡片系统
- `.chart-section` / `.chart-header` / `.chart-title` 图表区域
- `.rank-badge` / `.rank-badge.gold` 排名徽章
- `TrendLineChart` ECharts曲线图组件
