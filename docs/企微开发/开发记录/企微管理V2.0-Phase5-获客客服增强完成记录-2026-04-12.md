## JSON 规范遵循情况

全部代码严格遵循 `.dev-rules.md` JSON 规范：

1. ✅ JSON 字段存储使用 `JSON.stringify()` 序列化（weightConfig, dailyStats, servicerUserIds）
2. ✅ JSON 字段读取使用 `safeJsonParse()` 安全反序列化
3. ✅ API 请求体由 axios 自动序列化，未手动拼接
4. ✅ 前端 `JSON.parse()` 包裹 try-catch
5. ✅ 所有 API 响应使用 `res.json()` 标准格式
