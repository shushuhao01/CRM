## 五、功能详细说明

### 5.1 租户授权管理（TenantAuth.vue）

- **列表功能**：展示所有wecom_configs记录，含租户关联信息
- **搜索过滤**：按租户名/企业ID关键词 + 授权类型(self_built/third_party) + 连接状态
- **详情弹窗**：完整配置信息 + 授权范围JSON + 关联数据统计(客户/绑定/消息)
- **操作**：停用(revoke) / 恢复(restore)，含原因输入确认

### 5.2 套餐模板管理（PackageTemplate.vue）

- **模板CRUD**：套餐名称、人数、时长、售价/原价、折扣自动计算
- **功能特性**：Tag标签动态添加/删除
- **排序与启停**：sortOrder排序 + isEnabled启停用开关
- **兼容存储**：优先写入wecom_vas_configs表，表不存在时降级到system_config JSON

### 5.3 租户套餐管理（TenantPackage.vue）

- **列表展示**：套餐类型(Tag) + 席位用量(进度条) + 状态 + 到期日期（到期变红+即将到期变黄）
- **变更操作**：套餐类型/人数/存储方式/到期日期一站式修改
- **续费操作**：输入续费月数，基于当前到期日智能叠加

### 5.4 资源配额监控（QuotaMonitor.vue）

- **总体仪表盘**：Dashboard环形图展示整体使用率
- **6项关键指标**：活跃租户/总席位/已使用/总消息/总存储/使用率
- **用量排行 TOP 20**：按使用率降序，含进度条可视化
- **即将到期**：30天内到期的租户列表，区分严重程度(7天内红色/30天内黄色)
- **超限告警**：实际使用人数 > 最大人数的租户列表

### 5.5 增值服务配置增强（VasConfig.vue）

- **续费折扣**：可配置多档续费折扣（如6个月95折/12个月9折/24个月8折）
- **预设套餐包**：可配置前端购买弹窗展示的套餐包（名称/人数/价格/标签）
- **跳转链接**：引导前往套餐模板管理页面进行更精细管理

### 5.6 企微系统配置（WecomSystemConfig.vue）

5大配置区域：
1. **全局配置**：启用开关、默认授权类型、单租户最大配置数、API速率限制
2. **会话存档默认配置**：保留天数、存储方式、同步间隔、批量大小、媒体下载
3. **OSS存储配置**：云服务商、Region/Bucket/Endpoint/Key
4. **通知告警配置**：到期提醒天数(多档)、配额告警阈值、告警邮箱/Webhook
5. **限流配置**：全局/单租户分别的最大请求数和时间窗口

## 七、JSON规范遵循

- 所有后端API使用 `res.json()` 返回响应，无手动 `JSON.stringify` 拼接
- JSON字段(authScope/authCorpInfo/authUserInfo)使用 `try { JSON.parse() } catch { null }` 安全解析
- 套餐模板存储使用 `JSON.stringify(configValue)` 序列化
- 前端表单数据通过 `request.post/put` 提交，axios自动处理序列化
- WecomSystemConfig保存时使用 `JSON.parse(JSON.stringify(form))` 深拷贝后提交
- 默认配置通过函数 `getDefaultPackageTemplates()` / `getDefaultSystemConfig()` 返回纯对象
