## 一、项目概述

根据《企微服务商应用开发全流程分析与实施方案》，开发企微服务商H5独立应用。

- **前端**: Vue3 + Vite + Vant4 + Pinia + TypeScript
- **后端**: 融入现有CRM后端，新增 `h5-auth.ts`、`h5-app.ts`、`h5-sidebar.ts` 路由模块
- **原则**: 不修改已有侧边栏H5代码，不引入新错误

## 四、技术要点

1. **鉴权复用**: H5使用与侧边栏相同的 `authenticateSidebarToken` 中间件，JWT payload中type为`sidebar`
2. **双模式Token**: 通过 `WecomTokenService.getAccessToken(config)` 自动适配自建/第三方应用模式
3. **安全加固**: 所有接口都有限流中间件，绑定接口有额外的速率限制
4. **数据脱敏**: 客户手机号在返回前自动脱敏 (138****5678)
5. **静态托管**: 生产环境通过 `app.use('/h5', express.static(...))` 服务H5前端

## 五、部署说明

### 开发环境

```bash
cd h5
npm install    # 已完成
npm run dev    # 启动开发服务器 http://localhost:5174
```

Vite开发服务器会通过proxy将API请求代理到后端 `http://localhost:3000`。

### 生产环境

```bash
cd h5
npm run build  # 构建到 h5/dist/
```

后端 `app.ts` 已配置静态文件托管，构建产物将通过 `/h5` 路径提供服务。
