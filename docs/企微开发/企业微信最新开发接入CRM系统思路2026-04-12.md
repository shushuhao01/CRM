## 一、项目概述

### 1.1 项目目标

将CRM SaaS系统接入企业微信，成为企业微信官方服务商，通过第三方应用模式为租户企业提供企微客户通讯与私域运营能力。本方案覆盖四个端口的完整开发设计：

- **服务商端**：在企业微信服务商后台完成应用创建、权限配置、上线发布
- **CRM租户端**：租户登录CRM后使用的企微管理功能模块
- **运营管理后台**：你作为SaaS运营方使用的后台，管理租户、套餐、监控
- **会员中心**：租户在官网登录后自助查看、购买、管理企微服务的门户

### 1.2 应用类型说明

建议创建的应用类型为 **「营销获客」** 应用。该类型应用可整合获客助手、微信客服、客户联系等全套获客相关API，支持无需应用主页的轻量级接入，默认不在客户企业的工作台展示。

## 二、服务商端：企业微信平台配置

### 2.1 服务商注册与认证

**步骤1：注册服务商账号**
1. 访问企业微信服务商官网，注册服务商账号
2. 完成企业认证（需提供营业执照等资质）
3. 认证费用：300元/年（小型企业）或3000元/年（大型企业）

**步骤2：创建代开发应用模板**
1. 登录服务商后台，点击左侧导航栏「应用代开发」菜单
2. 点击「创建代开发应用模板」按钮
3. 填写应用基本信息：
    - 应用名称：如“XX CRM企微助手”
    - 应用Logo：上传应用图标（建议750×750px）
    - 应用简介：描述应用功能（不超过120字）
    - 应用分类：选择「营销获客」

### 2.2 应用权限配置

在应用配置页面的「应用权限」模块，根据产品功能需求勾选以下权限项：

| 权限类别 | 权限项 | 用途说明 | 是否必须 |
|---------|--------|---------|---------|
| **通讯录** | 通讯录只读 | 同步企业组织架构 | ✅ 必须 |
| **客户联系** | 客户管理、客户群管理、客户朋友圈 | 管理外部联系人、客户群、发布朋友圈 | ✅ 必须 |
| **客户联系** | 获客助手 | 创建和管理获客链接 | ✅ 必须 |
| **客户联系** | 建联客户信息 | 获取客户添加流水、消息次数 | ✅ 必须 |
| **微信客服** | 管理账号、分配会话和收发消息 | 客服账号管理和消息收发 | ✅ 必须 |
| **数据与智能专区** | 企业会话内容（会话存档） | 获取会话记录用于存档分析 | ⚠️ 按需 |
| **对外收款** | 对外收款记录查询 | 获取收款记录 | ⚠️ 按需 |
| **基础API** | 身份验证、消息推送 | 用户身份校验和应用消息推送 | ✅ 必须 |
| **JS-SDK** | 网页授权、客户端API | 侧边栏H5页面调用企微能力 | ✅ 必须 |

> **重要**：针对每一项权限，需提交用途说明（场景描述+场景示意图），审核通过后方可生效。用途说明也将展示在企业的数据权限授权页。

### 2.3 回调URL配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **指令回调URL** | 接收企业微信推送的事件通知 | `https://api.yourcrm.com/wecom/callback` |
| **数据回调URL** | 接收数据API相关回调 | `https://api.yourcrm.com/wecom/data-callback` |
| **Token** | 回调验证Token（自定义字符串） | 随机生成32位字符串 |
| **EncodingAESKey** | 消息加解密密钥 | 随机生成43位字符串 |

**需要监听的回调事件**：
- `create_auth`：企业授权应用（获取永久授权码）
- `cancel_auth`：企业取消授权
- `change_external_contact`：客户变更事件（添加/删除/编辑）
- `change_external_chat`：客户群变更事件
- `kf_msg_or_event`：微信客服消息事件
- `customer_acquisition`：获客助手相关事件

### 2.4 可信域名配置

1. 进入应用详情页，点击「网页授权及JS-SDK」
2. 设置可信域名：`yourcrm.com`（你的CRM系统主域名）
3. 下载验证文件并上传至域名根目录完成验证
4. 配置企业可信IP：添加你的后端服务器出口IP

### 2.5 应用上线

1. 在服务商后台选择「应用管理 → 代开发应用上线」
2. 提交审核（需提供测试企业ID供官方测试）
3. 审核通过后，应用状态变更为「已上线」
4. 在服务商后台获取应用二维码，嵌入CRM的授权页面供租户扫码

### 3.2 二级菜单详细设计

#### 3.2.1 企业授权

**页面功能**：管理本企业与企微的授权状态，是所有功能的前提。

| 功能模块 | 说明 |
|---------|------|
| **授权状态卡片** | 显示当前授权状态：未授权/已授权/已过期/授权范围；显示企业名称、授权时间、到期时间、已授权权限列表 |
| **扫码授权区** | 展示授权二维码。租户管理员使用企业微信扫码后，在手机上确认授权并设置应用可见范围 |
| **权限申请引导** | 对于数据API（获客数据、会话存档等），授权后需二次授权。展示“待申请”状态的权限项，点击调用JSAPI唤起授权页 |
| **可见范围设置** | 设置应用在企微内的可见范围（哪些部门/成员可以使用）。调用「设置应用可见范围」API |
| **授权记录** | 展示历史授权操作记录（授权时间、操作人、状态变化） |

**关联API**：
- 获取授权二维码（服务商后台生成）
- 接收授权成功回调 → 保存`permanent_code`
- 获取企业授权信息
- 调用JSAPI唤起数据权限授权页
- 设置应用可见范围

**技术要点**：
- 授权成功后，企业微信会向回调URL推送`create_auth`事件，包含`AuthCode`
- 后端用`AuthCode`换取`permanent_code`（永久授权码）并加密存储
- 后续所有API调用使用`permanent_code`换取`access_token`
- 数据API必须经过企业管理员的二次授权方可生效，授权有效期90天

#### 3.2.2 通讯录

**页面功能**：同步和管理企业微信组织架构。

| 功能模块 | 说明 |
|---------|------|
| **部门结构树** | 树形展示企微部门架构，支持展开/折叠，显示部门名称、成员数量 |
| **成员列表** | 表格展示选中部门下的成员，含姓名、职位、手机号、邮箱、企微头像 |
| **同步操作** | 手动同步（立即从企微拉取最新组织架构）、定时同步（配置每日同步时间） |
| **成员详情** | 点击成员查看详细信息，含企微userid、open_userid、对外信息等 |
| **搜索** | 按姓名、手机号、部门搜索成员 |

**关联API**：
- 获取部门列表
- 获取部门成员详情
- 获取成员详细信息

**注意事项**：
- 第三方应用仅可访问应用可见范围内的成员和部门
- 建议每日凌晨执行一次全量同步，确保数据一致性

#### 3.2.3 客户管理

**页面功能**：管理企业微信外部联系人（客户）。

| 功能模块 | 说明 |
|---------|------|
| **客户列表** | 表格展示所有外部联系人，含客户昵称、头像、所属员工、添加时间、客户类型（微信/企微） |
| **客户筛选** | 按添加时间、所属员工、标签、跟进状态等条件筛选 |
| **客户搜索** | 按昵称、备注名、手机号搜索 |
| **客户详情** | 点击查看详情：基本信息（昵称、头像、性别、地区）、企业信息（公司、职位）、添加来源、互动轨迹 |
| **标签管理** | 创建/编辑企业标签和标签组（单个企业最多10000个标签），支持批量/手动为客户打标签（每人最多3000个标签） |
| **客户备注** | 编辑客户备注名、描述、企业信息、手机号等 |
| **客户画像** | 整合标签、备注、互动记录、订单等数据形成客户统一画像 |
| **批量操作** | 批量打标签、批量分配员工、批量导出 |

**关联API**：
- 获取客户列表
- 获取客户详情
- 批量获取客户详情
- 编辑客户备注信息
- 管理企业标签（创建/编辑/删除）
- 为客户打标签/移除标签
- 获取客户互动轨迹

#### 3.2.4 获客管理

**页面功能**：创建和管理获客链接，配置加粉权重，分析渠道效果。

| 功能模块 | 说明 |
|---------|------|
| **获客链接列表** | 展示所有创建的获客链接，含链接名称、状态（启用/停用）、包含成员数、今日添加数、累计添加数、创建时间 |
| **创建链接** | 弹窗创建新链接：填写链接名称、选择包含的成员/部门、设置渠道参数（来源标识）、是否跳过验证 |
| **编辑链接** | 修改链接名称、调整成员列表、启用/停用 |
| **删除链接** | 删除不再使用的链接（删除后无法恢复） |
| **链接详情** | 点击查看详情：链接配置信息、成员列表及每人分配客户数、链接URL和二维码（可复制/下载） |
| **加粉权重配置** | 为成员/部门配置权重值（如1-10），高权重成员优先分配客户。系统通过动态调整获客链接中的成员列表实现 |
| **渠道分析** | 按获客链接统计：添加客户数、流失数、转化率；按日期趋势图展示；支持导出报表 |
| **使用量监控** | 展示获客助手使用量（已用次数/总配额），剩余不足时告警提示 |

**关联API**：
- 创建获客链接
- 获取获客链接列表
- 获取获客链接详情
- 编辑获客链接
- 删除获客链接
- 获取获客客户列表
- 获取获客链接添加流水

**技术要点**：
- 每个获客链接最多可配置500名成员，客户点击后系统自动平均分配
- 已有好友关系的客户会自动分配给原有员工，避免重复添加
- 加粉权重功能通过定时任务（如每分钟）调用“编辑获客链接”接口动态调整成员列表实现
- 调用频率限制：获客链接相关API需控制调用频率，避免触发限频

#### 3.2.5 客户群管理

**页面功能**：管理企业微信外部客户群。

| 功能模块 | 说明 |

- 获取客户群列表
- 获取客户群详情

#### 3.2.6 微信客服

**页面功能**：管理微信客服账号，查看会话记录，统计客服数据。

| 功能模块 | 说明 |
|---------|------|
| **客服账号列表** | 展示所有客服账号，含账号名称、接待人员、在线状态、今日会话数 |
| **创建/编辑账号** | 创建新客服账号或编辑现有账号：账号名称、头像、接待人员配置 |
| **接待人员管理** | 为每个客服账号配置接待成员/部门 |
| **会话记录** | 查看客服会话历史：客户昵称、接待客服、会话时间、消息数、满意度评价 |
| **会话详情** | 点击会话查看完整聊天记录（文本、图片、文件等消息类型） |
| **数据统计** | 统计客服数据：总会话数、平均响应时长、满意度评分分布、客服工作量排行 |
| **快捷回复库** | 配置客服快捷回复话术库（企业话术/个人话术） |

**关联API**：
- 管理客服账号（创建/编辑/删除/列表）
- 同步客服消息
- 发送客服消息
- 获取客服会话统计

**技术要点**：
- 微信客户发送消息时，企微后台会推送`kf_msg_or_event`回调事件，包含token
- 收到回调后，调用「同步消息」接口获取具体的消息内容和事件
- 支持的消息类型：文本、图片、语音、视频、文件、位置、链接、名片、小程序、菜单、事件

#### 3.2.7 侧边栏配置

**页面功能**：配置企业微信聊天侧边栏（聊天工具栏）的H5页面。

| 功能模块 | 说明 |

| **添加工具栏** | 弹窗配置：页面名称、H5页面URL（需在可信域名下）、使用场景（单聊/群聊/均支持）、排序 |
| **编辑/删除** | 修改工具栏配置或删除 |
| **预览** | 生成预览链接，可在浏览器中预览侧边栏页面效果 |
| **JS-SDK配置说明** | 展示可信域名配置状态、企业可信IP配置状态；提供JS-SDK初始化代码示例 |

**关联API**：
- 配置聊天工具栏菜单
- 获取聊天工具栏菜单

**技术要点**：
- 侧边栏页面为H5页面，需部署在你的CRM系统可信域名下
- 页面需引入企业微信JS-SDK，通过`ww.register`注册身份信息
- 侧边栏仅在“外部联系人”和“外部联系群”的对话中可打开
- 可通过JS-SDK调用企微客户端能力：获取当前客户ID、打开客户详情页、发送消息等

## 四、会话存档专项设计（三级标签页）

会话存档是CRM系统的核心数据能力模块，采用**三级标签页**结构组织功能。

**菜单层级**：
```
企微管理
└── 会话存档
    ├── 标签页1：会话聊天
    ├── 标签页2：敏感词
    ├── 标签页3：质检规则
    └── 标签页4：设置
```

### 4.1 标签页1：会话聊天

**页面布局**：左侧联系人列表 + 右侧会话记录展示（UI参考微信）

**左侧面板 - 联系人/群聊列表**：

| 功能模块 | 说明 |
|---------|------|
| **搜索框** | 按员工姓名、客户昵称、群名称搜索 |
| **分类Tab** | 三个Tab切换：全部会话、员工维度（按员工分组展示其客户/群）、客户维度（直接展示所有客户） |
| **会话列表项** | 每个列表项显示：头像（员工/客户/群）、名称、最后一条消息预览、最后消息时间、未读标记 |
| **选中态** | 点击列表项，右侧加载该会话的完整聊天记录 |

**右侧面板 - 聊天记录展示**：

| 功能模块 | 说明 |
|---------|------|
| **会话头部** | 显示当前会话对象名称、成员信息（单聊）/群成员数量（群聊） |
| **消息列表区域** | 按时间倒序或正序展示消息气泡：支持文本、图片、文件、语音（可播放）、视频、撤回消息（显示“已撤回”）等消息类型。文本消息支持长按复制 |
| **日期分隔线** | 按日期分组展示消息，显示“今天”、“昨天”、“YYYY-MM-DD”分隔线 |
| **消息搜索** | 在当前会话内搜索关键词，高亮匹配结果，支持上下翻页 |
| **时间筛选** | 按日期范围筛选展示该时间段内的消息（快捷选项：今天、最近3天、最近7天、自定义） |
| **消息导出** | 导出当前会话的聊天记录为PDF或Excel |
| **加载更多** | 滚动到顶部自动加载历史消息（分页加载，每页1000条） |

**关联API**：
- 获取会话记录（sync_msg）
- 获取媒体文件（图片/文件/语音）

**技术要点**：
- 调用会话存档API前需先调用「设置公钥」接口，配置RSA公钥用于消息解密
- 会话记录API可获取5天内的企业客户会话记录数据，一次拉取上限1000条，调用频率不超过600次/分钟
- 支持三种拉取模式：默认模式、预分页模式（高吞吐）、自适应模式
- 获取的消息为加密状态，需用私钥解密后展示
- 媒体文件（图片、文件、语音）需单独调用接口下载并展示

### 4.2 标签页2：敏感词

**页面功能**：配置敏感词库，对会话内容进行敏感词检测和告警。

| 功能模块 | 说明 |
|---------|------|
| **敏感词库管理** | 表格展示所有敏感词，含敏感词内容、分类（涉黄/涉政/暴恐/竞品/自定义）、等级（提示/告警/拦截）、状态 |
| **添加敏感词** | 弹窗添加：输入敏感词、选择分类、设置等级、是否启用 |
| **批量导入** | 支持Excel模板批量导入敏感词 |
| **敏感词分类管理** | 管理敏感词分类标签，支持新增/编辑/删除 |
| **检测记录** | 表格展示触发的敏感词记录：触发时间、员工、客户/群、敏感词、等级、会话上下文 |
| **告警通知** | 配置告警规则：触发敏感词后通过站内信/短信/企微消息通知指定管理员 |
| **白名单** | 配置免检员工/客户白名单 |

### 4.3 标签页3：质检规则

**页面功能**：配置会话质检规则，对员工服务质量进行自动评分和质检。

| 功能模块 | 说明 |
|---------|------|
| **质检规则列表** | 展示所有质检规则，含规则名称、类型（响应时长/消息数/关键词/情绪）、状态 |
| **创建规则** | 弹窗配置：规则名称、规则类型、条件参数、分值、适用员工范围 |
| **规则类型支持** | 响应时长（如超过5分钟回复扣分）、消息数（单日会话数低于X条）、关键词命中（如未发送结束语）、客户情绪（负面情绪检测） |
| **质检评分** | 自动计算会话质量评分，按员工/部门/时间统计 |
| **质检报告** | 生成质检报告：员工得分排行、低分会话列表、改进建议 |
| **人工复核** | 支持管理员对系统评分进行人工复核和调整 |

### 4.4 标签页4：设置

**页面功能**：配置会话存档的基础参数。

| 功能模块 | 说明 |
|---------|------|
| **开通引导** | 若租户未开通数据与智能专区，展示开通引导步骤：企业微信后台开通 → 设置公钥 → 完成授权 |
| **公钥配置** | 上传RSA公钥文件或粘贴公钥内容，用于企微加密会话消息。点击“测试连接”验证配置是否正确 |
| **授权状态** | 展示数据与智能专区授权状态（已授权/未授权/已过期）、授权范围（成员列表）、授权到期时间（90天有效期） |
| **续期操作** | 授权到期前7天展示提醒，点击“申请续期”调用JSAPI唤起授权续期页 |
| **存储配置** | 显示当前存储使用量/配额、存储天数（按套餐配置）；配置数据清理策略 |
| **拉取设置** | 配置会话存档拉取模式（默认/预分页/自适应）、拉取频率（实时/每5分钟/每小时） |
| **成员授权范围** | 设置需要存档的成员范围，仅范围内成员的会话会被拉取 |

**技术要点**：
- 数据API授权有效期为90天，到期前7天会通过应用推送续期提醒
- 授权过期后将无法获取会话数据，需在临期前引导管理员完成续期
- 企业管理员可随时在企微后台撤回已授予的数据权限，需在代码中处理“权限随时丢失”的容错机制

## 十、关键注意事项

### 10.1 权限与授权
- 新安装应用的企业有**90天免费试用期**，试用期内无需购买接口许可即可调用API
- 数据API必须经过企业管理员的**二次授权**方可生效，授权有效期90天，到期前7天需引导续期
- 企业管理员可随时在企微后台撤回已授予的权限，代码需具备容错机制

### 10.2 API调用限制
- 会话存档API：单次拉取上限1000条，调用频率不超过600次/分钟
- 获客链接API：每个获客链接最多可配置500名成员
- 客户标签：每个成员对单个客户最多打3000个标签，单个企业最多配置10000个标签

### 10.3 数据安全
- 获取的数据需进行加密存储，严禁将原始数据转卖或泄露给第三方
- 会话存档消息需用RSA私钥解密后展示，公钥配置在企微后台
- 租户间数据必须绝对隔离（所有查询强制注入tenant_id）

### 10.4 费用相关
- 接口许可费：1-50元/人/年，按企业维度累进阶梯计费
- 会话存档：服务版约450元/账号/年，企业版约900元/账号/年
- 获客助手：1元/次（成功添加客户），每企业免费体验50次
- 认证费：300元/年（小型企业）
