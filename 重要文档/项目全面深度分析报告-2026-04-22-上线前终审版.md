## 一、项目全景

### 1.1 产品定位

云客CRM是一个 **SaaS + 私有部署双模式** 的完整CRM商业产品体系，包含5个子系统 + 会员中心模块：

| 子系统 | 技术栈 | 文件数（最新统计） | 端口 | 定位 |
|--------|--------|--------|------|------|
| **Backend** | Node.js + Express + TypeORM + MySQL | **435个TS文件** | 3000 | 统一后端API服务 |
| **CRM前端** | Vue 3 + TS + Element Plus + Pinia | **373个Vue + 178个TS** | 5173 | 主业务系统（员工使用） |
| **Admin后台** | Vue 3 + TS + Element Plus + Axios | **96个Vue + 15个TS** | 5174 | 平台管理后台（运营方使用） |
| **Website官网** | Vue 3 + TS + 纯手写SCSS | **52个Vue + 11个TS** | 8080 | 产品展示 + 注册 + 支付 + 会员中心 |
| **crmAPP** | uni-app + Vue 3 + TS | **18个Vue + 16个TS** | -- | 电销外呼手机APP |

**全项目代码总量：约 450,000+ 行**

### 1.2 商业闭环

```
潜在客户访问官网 --> 选择套餐 --> 注册/支付（含订阅付费）--> 自动开通租户 --> 下发登录信息
                                                                          |
管理后台运营管理 <-- 授权/续费/模块控制/扩容 <-- 管理员 <-- 分配账号 <-- 登录CRM系统
                                                                   |
                                         +-------------------------+-------------------------+
                                         |                         |                         |
                                移动端APP          CRM日常使用                  会员中心
                               外呼/录音/跟进    客户/订单/物流/业绩          续费/扩容/账单
                                         |         企微/虚拟商品/AI           短信额度/企微服务
                                         +-------------------------+-------------------------+
```
